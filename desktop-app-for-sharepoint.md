# <a name="desktop-app-for-sharepoint"></a>Aplikasi desktop untuk SharePoint

Organisasi yang dimasukkan dalam program pribadi pratinjau untuk Desktop App untuk SharePoint. Microsoft saat ini tidak mendukung aplikasi Desktop untuk SharePoint (pribadi Preview). Silakan gunakan Mitra Microsoft pribadi pratinjau bintara Anda untuk mendapatkan bantuan dengan aplikasi Desktop untuk SharePoint.
