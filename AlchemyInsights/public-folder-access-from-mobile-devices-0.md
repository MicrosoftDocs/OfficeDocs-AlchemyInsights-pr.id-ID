# <a name="public-folder-access-from-mobile-devices"></a>Akses folder publik dari perangkat seluler

Folder publik tidak dapat diakses dari perangkat mobile.
  
Jika Anda ingin memiliki fungsi yang mirip dengan folder publik dan mengaksesnya dari perangkat seluler, lihat [Pelajari tentang kantor 365 kelompok](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) untuk alternatif yang lebih baik.
