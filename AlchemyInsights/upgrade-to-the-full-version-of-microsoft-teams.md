# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Memutakhirkan ke versi lengkap Microsoft teams

Perlu memutakhirkan dari versi penuh Microsoft teams gratis? Lihat [memutakhirkan Microsoft teams gratis ke versi langganan microsoft 365](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Anda harus memenuhi persyaratan berikut:

- Anda adalah orang yang mendaftar untuk tim versi gratis saat ini.
- Jika Anda memiliki domain Anda sendiri, itu belum dapat dikaitkan dengan Azure Active Directory.
- Untuk mentransfer data saat memutakhirkan, Anda harus berlangganan selama proses pemutakhiran. Jika Anda membeli langganan di luar proses pemutakhiran, Anda tidak bisa mentransfer data.
