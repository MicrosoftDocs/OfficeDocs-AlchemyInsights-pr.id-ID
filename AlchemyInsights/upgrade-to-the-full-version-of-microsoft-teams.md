# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Tingkatkan ke versi lengkap Microsoft teams

Perlu untuk meng-upgrade dari gratis ke versi lengkap dari Microsoft teams? Lihat [meningkatkan Microsoft teams gratis ke microsoft 365 versi langganan](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Anda harus memenuhi persyaratan ini:

- Anda adalah orang yang mendaftar untuk tim versi gratis saat ini.
- Jika Anda memiliki domain sendiri, itu tidak dapat dikaitkan dengan Azure Active Directory.
- Untuk mentransfer data saat Anda meng-upgrade, Anda harus berlangganan selama proses upgrade. Jika Anda membeli langganan di luar proses upgrade, Anda tidak dapat mentransfer data.
