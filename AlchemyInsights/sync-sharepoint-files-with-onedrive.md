# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>Menyinkronkan file SharePoint dengan klien sinkronisasi OneDrive yang baru

<p>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Perintah Buka dengan Explorer akan membuka instans lokal Windows Explorer yang menampilkan struktur folder di server yang menghosting situs SharePoint. Namun demikian, kami menyarankan untuk <a href="https://support.office.com/id-ID/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88">menyinkronkan file SharePoint dengan klien sinkronisasi OneDrive yang baru</a> yang menyediakan <a href="https://support.office.com/id-ID/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e">File Sesuai Permintaan</a> karena menyediakan akses lokal ke file Anda dan menawarkan performa terbaik</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Jika Anda memilih untuk menggunakan tampilan Explorer, bukan menggunakan klien sinkronisasi baru, pastikan Anda mengikuti langkah-langkah dan praktik terbaik dalam artikel di bawah ini.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/id-ID/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4">Cara menggunakan perintah "Buka dengan Explorer" untuk menyelesaikan masalah di SharePoint Online</a></span></li> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/id-ID/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2">Menyalin atau memindahkan file pustaka menggunakan Buka dengan Explorer</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Catatan:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> Tombol Buka dengan Explorer tidak muncul dalam pengalaman pustaka yang baru. Klik menu menurun<strong>Tampilan</strong> di bagian kanan atas (nama menu menurun berubah tergantung pada tampilan Anda saat ini), lalu klik <strong>Tampilkan</strong> di File Explorer.</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Catatan:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Buka dengan Explorer di SharePoint menggunakan <strong>kontrol ActiveX</strong>, sehingga hanya didukung di Internet Explorer 10 atau 11. Buka dengan Explorer tidak berfungsi di Windows dengan Microsoft Edge, Google Chrome, Mozilla Firefox, atau di platform Mac. Karena itu, opsi &ldquo;Tampilan Explorer&rdquo; mungkin berwarna abu-abu.</span></p> <ul> <li>
