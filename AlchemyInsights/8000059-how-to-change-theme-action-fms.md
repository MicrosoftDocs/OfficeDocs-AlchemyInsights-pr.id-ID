# <a name="how-to-change-your-theme"></a>Bagaimana untuk mengubah tema

Untuk mengubah tema Anda di Outlook.com atau Outlook di web, pergi ke **pengaturan**![pengaturan](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) dan pilih tema **tema**. Untuk melihat semua tema yang tersedia, pilih **Lihat semua** **tema**.
