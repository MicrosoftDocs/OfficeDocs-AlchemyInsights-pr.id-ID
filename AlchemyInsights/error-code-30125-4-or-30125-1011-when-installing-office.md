# <a name="solutions-for-error-30125-4-or-30125-1011"></a>Solusi untuk kesalahan 30125-4 atau 30125-1011

Anda mungkin melihat kesalahan ini jika Anda software anti-virus, firewall, pengaturan proxy atau koneksi mencegah Anda dari menginstal Office. Berikut adalah beberapa hal yang Anda dapat mencoba untuk memperbaiki masalah:
  
- Restart komputer Anda dan menginstal ulang kantor
    
- Menginstal Office menggunakan [kantor offline installer](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c.aspx)
    
- Untuk sementara menonaktifkan pengaturan proxy
    
- Sementara Nonaktifkan perangkat lunak antivirus
    
- Untuk sementara menonaktifkan firewall
    
Untuk informasi lebih lanjut lihat [kode kesalahan 30125-4 atau 30125-1011 ketika menginstal Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612.aspx)
