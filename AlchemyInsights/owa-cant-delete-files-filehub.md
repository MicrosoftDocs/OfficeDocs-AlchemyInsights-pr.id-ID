# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>Tidak dapat menghapus file atau foto dari file dalam Outlook di web

**File** memperlihatkan file dan foto yang dilampirkan ke pesan di kotak surat Anda. Untuk menghapus file dan foto dari file, Anda harus menghapus pesan email yang dilampirkan. Setelah menghapus pesan, Anda juga perlu menghapusnya dari folder Item terhapus.
