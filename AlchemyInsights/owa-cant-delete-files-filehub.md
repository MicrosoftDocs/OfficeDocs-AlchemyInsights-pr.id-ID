# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>Tidak dapat menghapus file atau foto dari file di Outlook di web

**File** menunjukkan file dan foto-foto yang dilampirkan ke pesan di kotak pesan. Untuk menghapus file dan foto dari file, Anda perlu untuk menghapus pesan email mereka melekat. Setelah menghapus pesan, Anda juga perlu untuk menghapusnya dari folder Item dihapus.
