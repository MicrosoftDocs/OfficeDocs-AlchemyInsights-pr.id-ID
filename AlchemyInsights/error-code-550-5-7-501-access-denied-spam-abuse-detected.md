# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 Akses ditolak, penyalahgunaan spam terdeteksi

Biasanya, pesan ini terjadi saat pengguna mengirim pesan email dari alamat IP yang menggunakan domain *. onmicrosoft.com* awal yang ditetapkan untuk penyewa baru di Microsoft 365. Cara termudah untuk mengatasi masalah ini adalah:

1. [Tambahkan domain ke penyewa Anda](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain).

2. [Ubah Alamat email primer pengguna](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address) ke domain kustom baru yang baru saja ditambahkan.
