# <a name="configure-multi-factor-authentication-in-office-365"></a>Mengkonfigurasi otentikasi multi faktor di Office 365

Untuk mengkonfigurasi MFA, lihat [mengatur otentikasi multi faktor untuk Office 365 pengguna](https://docs.microsoft.com/office365/admin/security-and-compliance/set-up-multi-factor-authentication).

Perhatikan bahwa Anda harus menjadi admin global Office 365 untuk mengaktifkan dan mengkonfigurasi MFA.
