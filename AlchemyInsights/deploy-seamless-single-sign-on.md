# <a name="enable-seamless-sso"></a>Mengaktifkan SSO mulus

Azure Active Directory (Azure AD) mulus akses terusan (SSO akses mulus) otomatis masuk pengguna ketika mereka berada di desktop mereka perusahaan yang tersambung ke jaringan perusahaan Anda.
  
[Ikuti langkah-langkah dalam artikel ini untuk mengaktifkan SSO mulus](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
