# <a name="how-to-right-click-on-a-mac"></a>Bagaimana untuk klik kanan pada Mac

Pada komputer Mac, klik kanan dikenal sebagai sekunder klik atau Ctrl klik. Jika mouse Anda, sentuh, atau perangkat input lainnya tidak menyertakan klik kanan tombol atau cara lain untuk melakukan klik kanan, tahan **tombol Control** pada keyboard Anda ketika Anda klik. 
  
Untuk informasi lebih lanjut, pergi ke [Apple dukungan](https://go.microsoft.com/fwlink/?linkid=2022220&amp;clcid=0x409).
