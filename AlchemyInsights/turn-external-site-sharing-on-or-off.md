# <a name="turn-external-sharing-on-or-off-for-a-site"></a>Mengaktifkan atau menonaktifkan berbagi eksternal untuk situs

Anda harus merupakan admin global atau SharePoint untuk mengubah pengaturan berbagi eksternal untuk situs (sebelumnya disebut "kumpulan situs"). Pemilik situs tidak diperbolehkan untuk mengubah pengaturan ini. 

Untuk informasi selengkapnya, lihat [mengaktifkan atau menonaktifkan berbagi eksternal untuk situs](https://docs.microsoft.com/sharepoint/change-external-sharing-site).
