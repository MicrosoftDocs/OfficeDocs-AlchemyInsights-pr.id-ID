# <a name="export-contacts"></a>Ekspor kontak

1. Di bagian bawah halaman, pilih ikon orang untuk pergi ke halaman orang.
2. Pada toolbar, pilih **Kelola** > **ekspor kontak**. 
3. Memilih untuk mengekspor semua kontak atau hanya kontak dari folder tertentu, dan kemudian pilih **ekspor**.
4. Di bagian bawah halaman, pilih **Simpan** untuk menyimpan "contacts.csv" dalam map unduhan bawaan. Atau, pilih tanda panah di sebelah **Simpan** dan pilih **Simpan sebagai**.

Untuk selengkapnya, lihat [mengekspor kontak dari Outlook](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f#ID0EAACAAA=Outlook_on_the_web).
