# <a name="why-cant-i-edit-files"></a>Mengapa saya tidak dapat mengedit file?

Jika dokumen, lembar kerja, atau presentasi terkunci untuk mengedit dan tidak dapat dibuka di aplikasi desktop Windows, mungkin berkas yang telah diedit oleh orang lain, atau mereka memiliki berkas diperiksa. Mari kita lihat sekilas beberapa alasan [mengapa Anda mungkin tidak dapat mengedit file](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).

Jika Anda memiliki perpustakaan dengan **checkout, diperlukan** atau **validasi** kolom atau metadata, atau ketika **konsep keamanan item** diatur ke salah satu **hanya pengguna yang dapat mengedit** atau **hanya pengguna yang dapat menyetujui item** dalam versi pengaturan Perpustakaan:

- Untuk perpustakaan dokumen SharePoint, item ini akan disinkronkan sebagai baca-saja.

- Untuk Perpustakaan OneDrive, OneDrive akan menampilkan ikon kunci (ikon OneDrive file terkunci) di samping status sinkronisasi dan pengguna tidak akan dapat mensinkronisasi Perpustakaan hingga pengaturan yang disebutkan telah dihapus. 

Untuk informasi lebih lanjut mengenai file terkunci untuk Word atau Excel, silahkan referensi artikel di bawah ini.

- [File Excel terkunci untuk mengedit dengan pengguna lain](https://support.office.com/article/Excel-file-is-locked-for-editing-by-another-user-6fa93887-2c2c-45f0-abcc-31b04aed68b3)

- [Word file terkunci untuk mengedit dengan pengguna lain](https://support.microsoft.com/help/313472/the-document-is-locked-for-editing-by-another-user-error-message-when)
