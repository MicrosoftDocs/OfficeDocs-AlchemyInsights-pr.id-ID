# <a name="duplicate-object-synchronization-errors"></a>Duplikat kesalahan sinkronisasi objek

Jika Anda menerima salah satu galat berikut selama sinkronisasi:

- Sebuah objek disinkronisasi dengan alamat proxy yang sama sudah ada di direktori Layanan Online Microsoft.

- Pesan serupa untuk UserPrincipalName.

Menggunakan AADConnect baru kesehatan diagnostik untuk mengatasi kesalahan duplikat objek. Untuk informasi lebih lanjut, lihat [mendiagnosis dan memulihkan galat sinkronisasi atribut digandakan](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
