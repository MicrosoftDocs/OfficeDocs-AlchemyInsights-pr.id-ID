# <a name="restore-files-or-folders-from-the-sharepoint-recycle-bin"></a>Memulihkan file atau folder dari SharePoint recycle bin 

Ketika Anda menghapus item dari perpustakaan dokumen SharePoint atau daftar, itu tidak segera dihapus dari SharePoint. Item yang dihapus masuk ke Recycle Bin situs untuk jangka waktu tertentu. Pada saat itu, Anda dapat memulihkan item yang dihapus ke lokasi aslinya.

- [Memulihkan file atau folder dari SharePoint recycle bin](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be)
- [Kembalikan file atau folder yang dihapus di OneDrive](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)
