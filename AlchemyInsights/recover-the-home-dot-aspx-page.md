# <a name="recover-the-homeaspx-page"></a>Memulihkan Halaman Home. aspx

Jika situs halaman rumah dihapus, dapat dipulihkan dari Recycle Bin dengan mengakses recycle bin URL.

1. Navigasikan ke https:// \< tenantname>. SharePoint.com/Sites/nama \< situs>/_layouts/15/RecycleBin.aspx?, dengan mengganti <**tenantname**> dengan nama penghuni dan <**namename**> dengan judul website Anda.

2. Pilih halaman yang akan dipulihkan.

Jika halaman Beranda masih tidak ditemukan, Anda dapat membuat halaman situs baru dan [mengaturnya sebagai halaman beranda situs Anda](https://support.microsoft.com/en-gb/office/use-a-different-page-for-your-sharepoint-site-home-page-35a5022c-f84a-455d-985e-c691ab5dfa17?ui=en-us&rs=en-gb&ad=gb). 404 tidak ditemukan galat tidak lagi muncul ketika Anda mencoba untuk mengakses situs.
