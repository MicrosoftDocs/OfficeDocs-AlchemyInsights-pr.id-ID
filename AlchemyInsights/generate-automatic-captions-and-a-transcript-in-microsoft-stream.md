# <a name="generate-automatic-captions-and-a-transcript-in-microsoft-stream"></a>Menghasilkan teks otomatis dan transkrip di Microsoft Stream

Streaming dapat secara otomatis menghasilkan keterangan menggunakan teknologi pengenalan ucapan otomatis, dan Anda dapat melihat jendela transkrip bergulir agar mudah menavigasi dan mengikuti bersama dengan konten lisan video.

- [Hasilkan keterangan otomatis dan transkrip untuk video Microsoft Stream Anda](https://docs.microsoft.com/stream/portal-autogenerate-captions)

- [Menampilkan jendela transkrip bergulir untuk video di Microsoft Stream](https://docs.microsoft.com/stream/portal-configure-transcript-mode)
