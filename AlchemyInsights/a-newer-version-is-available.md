# <a name="delete-the-office-document-cache"></a>Menghapus Singgahan dokumen Office

Jika Anda menerima perintah Excel bahwa **versi file yang lebih baru tersedia**, coba hapus Singgahan dokumen Office dengan mengikuti instruksi di [Hapus singgahan dokumen Office Anda](https://support.office.com/article/b1d3765e-d71b-4bb8-99ca-acd22c42995d). Lalu Cobalah membuka kembali file tersebut.
