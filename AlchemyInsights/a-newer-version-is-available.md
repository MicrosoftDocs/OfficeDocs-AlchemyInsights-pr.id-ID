# <a name="delete-the-office-document-cache"></a>Menghapus Cache dokumen Office

Jika Anda menerima prompt Excel yang **versi terbaru dari file tersedia**, mencoba menghapus Cache dokumen Office dengan mengikuti petunjuk dalam [menghapus tembolok dokumen kantor](https://support.office.com/article/b1d3765e-d71b-4bb8-99ca-acd22c42995d). Kemudian mencoba membuka kembali file.
