# <a name="connecting-powershell-scripts-to-exchange-online"></a>Menyambungkan skrip PowerShell ke Exchange Online

Otentikasi dasar di Exchange Online akan menjadi usang, dan jalan ke depan adalah untuk menyambung menggunakan otentikasi berbasis sertifikat untuk skrip dan tugas tanpa pengawasan. Untuk mempelajari lebih lanjut, lihat [otentikasi hanya aplikasi untuk skrip tanpa pengawasan di modul EXO v2](https://docs.microsoft.com/powershell/exchange/app-only-auth-powershell-v2).
