# <a name="dashboard-doesnt-reflect-changes-after-refresh"></a>Dasbor tidak mencerminkan perubahan setelah refresh

Jika dasbor tidak mencerminkan perubahan setelah refresh, tunggu sekitar 10-15 menit agar refresh dicerminkan di ubin dasbor. Jika masih belum muncul, Re-pin visualisasi ke dashboard.

Untuk informasi selengkapnya, lihat [skenario penyegaran pemecahan masalah](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios).
