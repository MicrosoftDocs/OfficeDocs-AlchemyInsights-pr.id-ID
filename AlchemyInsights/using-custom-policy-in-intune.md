# <a name="using-custom-policy-in-intune"></a>Menggunakan kebijakan kustom dalam Intune

Intune memiliki banyak pengaturan built-in untuk perangkat Android, iOS, macOS dan Windows. Anda bisa juga membuat profil kustom, dan kemudian menyebarkan pengaturan kustom ini ke perangkat Anda menggunakan Intune. Profil kustom dirancang untuk menambahkan pengaturan perangkat dan fitur yang tidak dibangun di Intune.

[Pengaturan khusus untuk perangkat Android di Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Pengaturan khusus untuk perangkat Android perusahaan di Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Pengaturan khusus untuk perangkat iOS di Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Pengaturan untuk macOS perangkat di Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Pengaturan khusus untuk perangkat Windows 10 Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Pengaturan khusus untuk Windows holografik untuk bisnis perangkat di Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
