# <a name="using-custom-policy-in-intune"></a>Menggunakan kebijakan kustom di Intune

Intune memiliki banyak pengaturan untuk perangkat Android, iOS, macOS dan Windows. Anda juga dapat membuat profil kustom, lalu menyebarkan pengaturan kustom ini ke perangkat Anda menggunakan Intune. Profil kustom dirancang untuk menambahkan pengaturan perangkat dan fitur yang tidak disertakan dalam Intune.

[Pengaturan kustom untuk perangkat Android di Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Pengaturan kustom untuk perangkat Android Enterprise di Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Pengaturan kustom untuk perangkat iOS di Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Pengaturan kustom untuk perangkat macOS di Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Pengaturan kustom untuk perangkat Windows 10 di Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Pengaturan kustom untuk Windows Holographic untuk perangkat bisnis di Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
