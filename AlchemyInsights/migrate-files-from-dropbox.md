# <a name="migrate-files-from-dropbox"></a>Melakukan migrasi file dari Dropbox

Jika Anda melakukan migrasi dari Dropbox ke SharePoint online, Anda dapat mempertimbangkan penggunaan **Mover**.

Untuk informasi selengkapnya, lihat [mengesahkan konektor Dropbox](https://docs.microsoft.com/sharepointmigration/mover-dropbox).
