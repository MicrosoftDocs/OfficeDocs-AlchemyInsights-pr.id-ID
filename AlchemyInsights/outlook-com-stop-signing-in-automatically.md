# <a name="how-to-stop-signing-in-automatically"></a>Cara berhenti masuk secara otomatis

Untuk berhenti masuk ke Outlook.com secara otomatis, Hapus centang pada kotak agar **saya tetap masuk** di halaman tempat Anda memasukkan sandi.
  
Untuk mempelajari lebih lanjut, lihat [cara masuk atau keluar dari Outlook.com](https://support.office.com/article/e08eb8ac-ac27-49f4-a400-a47311e1ee7e?wt.mc_id=Office_Outlook_com_Alchemy).
