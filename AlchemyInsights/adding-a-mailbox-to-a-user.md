# <a name="adding-a-mailbox-to-a-user"></a>Menambahkan kotak mail ke pengguna

Untuk menambahkan kotak surat ke pengguna yang sudah dibuat atau pengguna baru, pengguna harus ditetapkan lisensi untuk Exchange Online. Jika pengguna telah ditetapkan lisensi, pilih Exchange Online di bawah bagian aplikasi di halaman lisensi untuk memulai proses untuk membuat kotak surat bagi pengguna. Untuk info lebih lanjut, lihat [menetapkan lisensi kepada pengguna](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).
