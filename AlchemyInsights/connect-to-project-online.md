# <a name="connect-to-project-online"></a>Hubungkan ke Project online

Anda dapat menyambung ke Project online dengan menggunakan:

- Klien Project online desktop untuk menyambung ke Project online di lingkungan Office 365. Untuk informasi lebih lanjut, lihat [menyambung ke Project online dengan klien desktop Project online](https://docs.microsoft.com/projectonline/connect-to-project-online-with-the-project-online-desktop-client).  
- Layanan Power BI untuk membuka wawasan dari Project Web App untuk membantu mengelola proyek, portofolio, dan sumber daya. Untuk informasi lebih lanjut, lihat [menyambung ke aplikasi web Project dengan Power BI](https://docs.microsoft.com/power-bi/service-connect-to-project-online).
