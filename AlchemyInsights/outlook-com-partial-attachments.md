# <a name="cant-preview-all-of-an-attachment"></a>Tidak dapat melihat pratinjau semua lampiran

Kami mengetahui masalah di mana beberapa pelanggan tidak dapat melihat pratinjau semua lampiran. Untuk mengatasinya, silakan buka lampiran di jendela baru.
