# <a name="public-folder-access-from-mobile-devices"></a>Akses folder publik dari perangkat seluler

Akses ke folder publik tidak berfungsi menggunakan perangkat seluler, silakan gunakan OWA (Outlook di web) atau Outlook untuk Windows Desktop atau Outlook untuk Mac klien untuk mengakses folder publik.

Jika Anda ingin memiliki fungsionalitas yang sama dengan folder publik dan memiliki konten yang dapat diakses di perangkat seluler, lihat [mempelajari tentang grup Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) untuk alternatif yang lebih baik.
