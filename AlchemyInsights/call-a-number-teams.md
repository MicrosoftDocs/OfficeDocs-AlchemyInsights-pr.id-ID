# <a name="call-a-phone-number-in-teams"></a>Menghubungi nomor telepon di teams

Masuk ke  **panggilan**, pilih  **Panggil nomor**, dan gunakan tombol angka untuk memasukkan angka. Lalu klik  **Panggil**.

Jika Anda tidak melihat opsi untuk menghubungi nomor, organisasi Anda mungkin tidak membeli paket panggilan. Tapi Anda masih dapat melakukan panggilan ke orang lain di teams.  

Untuk informasi selengkapnya, lihat [menghubungi nomor telepon dari teams](https://support.microsoft.com/office/20d24ace-2851-4c29-8441-30dd2a5cf078).
