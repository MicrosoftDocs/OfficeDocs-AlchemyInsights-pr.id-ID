# <a name="set-passwords-to-never-expire"></a>Mengatur password untuk tidak pernah berakhir

Mengatur password untuk tidak pernah berakhir dari [keamanan &amp; privasi](https://portal.office.com/adminportal/home#/settings/security). Klik "Edit" di samping standar kebijakan Password, dan mengubah slider ke "on".
  
Untuk informasi lebih lanjut lihat: [menetapkan kebijakan kadaluarsa sandi untuk organisasi Anda](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
