# <a name="how-to-close-your-desktop-analytics-account"></a>Cara menutup akun desktop Analytics

Jika Anda mengatur desktop Analytics di lingkungan Anda, dan kemudian memutuskan Anda perlu menghapusnya, gunakan proses di tautan berikut ini:

[Cara menutup akun](https://docs.microsoft.com/configmgr/desktop-analytics/account-close)

Langkah pertama adalah menghubungi dukungan Microsoft. Buka kasus dukungan untuk menutup akun desktop Analytics. Jangan melanjutkan langkah tambahan hingga Anda menerima konfirmasi bahwa Microsoft telah menutup akun Anda.
