# <a name="how-to-close-your-desktop-analytics-account"></a>Bagaimana menutup akun Desktop Analytics

Jika Anda mengkonfigurasi Desktop Analytics di lingkungan Anda, dan kemudian memutuskan Anda perlu menghapus itu, menggunakan proses di link berikut:

[Cara untuk menutup akun Anda](https://docs.microsoft.com/sccm/desktop-analytics/account-close)

Langkah pertama adalah untuk hubungi Microsoft Support. Membuka kasus dukungan untuk menutup akun Desktop Analytics. Jangan melanjutkan dengan langkah-langkah tambahan sampai Anda menerima konfirmasi bahwa Microsoft tutup akun Anda.
