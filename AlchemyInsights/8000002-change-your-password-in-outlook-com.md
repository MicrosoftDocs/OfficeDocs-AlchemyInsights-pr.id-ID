# <a name="change-your-password"></a>Mengubah sandi

Sandi Outlook.com adalah sama dengan sandi akun Microsoft.
  
Untuk mengubah sandi, pergi ke [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) dan pilih **Ubah sandi**. 
  
Jika Anda lupa password Anda, pergi ke [akun Microsoft reset password halaman](https://go.microsoft.com/fwlink/p/?linkid=841909).
  
Untuk selengkapnya, lihat [mengubah sandi Anda di Outlook.com](https://go.microsoft.com/fwlink/?linkid=873109).
