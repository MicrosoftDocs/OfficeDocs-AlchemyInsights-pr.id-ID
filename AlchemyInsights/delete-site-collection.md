# <a name="delete-a-sharepoint-site-collection"></a>Menghapus koleksi situs SharePoint

1. Di klasik SharePoint admin center, pada halaman situs koleksi, centang kotak di sebelah situs koleksi.
    
2. Pada tab koleksi situs, pilih **Hapus**.
