# <a name="delete-a-sharepoint-site"></a>Menghapus situs SharePoint

1. Di pusat admin SharePoint baru, pada halaman situs, klik kotak centang di sebelah situs yang ingin Anda hapus.
    
2. Di toolbar pita, klik **Hapus**.
