# <a name="troubleshoot-owner-issues"></a>Memecahkan masalah pemilik

Untuk memecahkan masalah terkait pemilik, lakukan langkah-langkah berikut:

1. [Menambahkan atau mengubah administrator langganan Azure](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-accessmanagement-managing-group-owners): grup Azure Active Directory (Azure AD) dimiliki dan dikelola oleh pemilik grup. Pemilik grup bisa menjadi pengguna atau kepala layanan, dan dapat mengelola grup, termasuk keanggotaan. Hanya pemilik grup atau administrator grup yang sudah ada yang dapat menetapkan pemilik grup. Pemilik grup tidak diwajibkan untuk menjadi anggota grup.
2. [Menambahkan atau mengubah administrator langganan Azure](https://docs.microsoft.com/azure/cost-management-billing/manage/add-change-subscription-administrator): artikel ini menjelaskan cara menambahkan atau mengubah peran administrator untuk pengguna yang menggunakan Azure RBAC di lingkup langganan.
3. Gunakan PowerShell untuk menambahkan pemilik grup atau pemilik aplikasi.
