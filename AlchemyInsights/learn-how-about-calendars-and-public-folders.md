# <a name="learn-about-using-public-folders-in-outlook"></a>Mempelajari tentang menggunakan folder publik di Outlook

Folder publik di Outlook memberi Anda tempat yang tepat untuk mengumpulkan, menata, dan berbagi informasi tentang topik atau proyek tertentu dalam organisasi Anda. Untuk informasi selengkapnya, lihat [membuat dan berbagi folder publik di Outlook](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97).
