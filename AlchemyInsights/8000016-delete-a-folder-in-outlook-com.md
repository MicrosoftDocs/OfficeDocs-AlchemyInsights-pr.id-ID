# <a name="delete-a-folder"></a>Hapus folder

1. Dalam pane kiri, klik kanan folder atau subfolder yang ingin Anda hapus dan pilih **Hapus**. 
    
2. Pilih **OK** untuk konfirmasi. 
    
Untuk selengkapnya, lihat [menghapus folder di Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
