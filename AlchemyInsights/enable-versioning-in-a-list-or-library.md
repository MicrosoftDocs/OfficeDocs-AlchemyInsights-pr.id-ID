# <a name="enable-versioning-for-a-list-or-library"></a>Mengaktifkan versi untuk daftar atau Perpustakaan

1. Buka daftar atau Perpustakaan.
    
2. Klik **pengaturan**, dan kemudian klik Setelan **Perpustakaan** atau **Daftar pengaturan**.
    
3. Klik **pengaturan versi**.
    
4. Pilih pengaturan yang Anda inginkan, dan kemudian klik **OK**.
