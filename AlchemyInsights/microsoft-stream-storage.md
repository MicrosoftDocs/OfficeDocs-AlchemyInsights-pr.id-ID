# <a name="microsoft-stream-storage"></a>Penyimpanan Microsoft Stream

Penyewa Microsoft Stream menerima alokasi tetap 500 GB penyimpanan dan tambahan 0,5 GB penyimpanan per pengguna berlisensi.
Untuk detail lebih lanjut tentang penyimpanan, lihat:

- [Ringkasan Microsoft Stream Storage](https://docs.microsoft.com/stream/license-overview#storage)
- [Tambahkan penyimpanan ke Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [Kuota dan keterbatasan di Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)
