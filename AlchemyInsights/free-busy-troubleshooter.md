# <a name="issues-seeing-calendar-freebusy-information"></a>Masalah melihat informasi bebas/sibuk kalender

Jika Anda tidak dapat melihat informasi bebas/sibuk kalender di kalender lain, atau jika orang lain tidak dapat melihat informasi bebas/sibuk di kalender Anda, gunakan [dukungan Microsoft dan asisten pemulihan](https://diagnostics.office.com/).
