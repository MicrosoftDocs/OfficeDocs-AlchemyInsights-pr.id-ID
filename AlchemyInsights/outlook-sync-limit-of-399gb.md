# <a name="outlook-sync-status-bar-remains-at-399-gb"></a>Outlook sync status bar tetap di 3.99 GB

Cobalah mengurangi ukuran konten folder atau membuat penyaring sinkronisasi untuk folder besar di atau dekat 4 GB.

Melihat Status bar kemajuan pernah menunjukkan lebih dari 3.99 GB tersisa pada awal [sinkronisasi folder yang besar untuk informasi lebih lanjut](https://support.microsoft.com/help/2738323/status-bar-progress-never-shows-more-than-3-99-gb-remaining-on-initial).
