# <a name="outlook-sync-status-bar-remains-at-399-gb"></a>Bilah status sinkronisasi Outlook tetap ada di 3,99 GB

Coba kurangi ukuran konten folder atau buat filter sinkronisasi untuk folder berukuran besar di atau dekat 4 GB.

Melihat perkembangan bilah status tidak pernah memperlihatkan lebih dari 3,99 GB yang tersisa pada [sinkronisasi awal folder besar untuk informasi selengkapnya](https://support.microsoft.com/help/2738323/status-bar-progress-never-shows-more-than-3-99-gb-remaining-on-initial).
