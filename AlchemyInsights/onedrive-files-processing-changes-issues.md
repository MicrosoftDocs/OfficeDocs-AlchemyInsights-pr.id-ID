# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: memecahkan masalah ' memproses perubahan '

Jika Anda melihat "pemrosesan perubahan" untuk jangka waktu yang lama, mungkin karena Anda memiliki file online terbuka, banyak file terjebak dalam antrian, atau sinkronisasi file yang sangat besar, lihat [OneDrive terjebak di pemrosesan](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Baca artikel ini untuk memecahkan masalah "pemrosesan perubahan." Anda juga akan melihat pesan ini jika Anda memiliki file yang berubah sepanjang waktu (seperti Outlook. File PST atau database). Pelajari cara [memindahkan file data Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) yang Anda gunakan secara rutin dari OneDrive.
