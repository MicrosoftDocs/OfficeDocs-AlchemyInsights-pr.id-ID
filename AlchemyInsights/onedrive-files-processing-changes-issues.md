# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: Memecahkan masalah 'Pengolahan perubahan'

Jika Anda melihat "Pengolahan perubahan" untuk jangka waktu, bisa jadi karena Anda memiliki online file terbuka, banyak file yang terjebak dalam antrian, atau sinkronisasi file sangat besar, melihat [OneDrive terjebak pada pengolahan](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Membaca artikel ini untuk memecahkan masalah "Pengolahan perubahan." Anda juga akan melihat pesan ini jika Anda memiliki file yang berubah sepanjang waktu (seperti Outlook. PST file atau database). Pelajari cara untuk [memindahkan file data Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) yang Anda gunakan secara teratur dari OneDrive.
