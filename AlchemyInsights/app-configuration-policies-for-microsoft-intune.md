# <a name="app-configuration-policies-for-microsoft-intune"></a>Kebijakan konfigurasi aplikasi untuk Microsoft Intune

Kebijakan konfigurasi aplikasi di Microsoft Intune menyediakan pengaturan konfigurasi untuk aplikasi iOS atau Android. Pengaturan konfigurasi ini memungkinkan aplikasi disesuaikan. Dukungan kebijakan konfigurasi dibatasi oleh versi sistem operasi dan aplikasi.

Untuk detail selengkapnya, lihat [kebijakan konfigurasi aplikasi untuk Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
