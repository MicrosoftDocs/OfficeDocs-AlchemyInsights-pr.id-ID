# <a name="learn-how-to-configure-agenda-settings"></a>Pelajari cara mengkonfigurasi pengaturan agenda

Agenda pengaturan terletak di pengaturan **kalender** di Outlook di web. Untuk selengkapnya, lihat [pengaturan kalender di Outlook di web](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).
  
Admin dapat menggunakan cmdlet **Set-CalendarNotification** di Exchange Online PowerShell untuk mengkonfigurasi pengaturan agenda. Untuk selengkapnya, lihat [Set-CalendarNotification](https://technet.microsoft.com/library/dd351284).
