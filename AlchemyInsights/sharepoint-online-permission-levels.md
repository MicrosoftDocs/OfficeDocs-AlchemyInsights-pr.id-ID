# <a name="sharepoint-online-permission-levels"></a>Tingkat izin SharePoint online

SharePoint dilengkapi dengan beberapa tingkat izin default yang dapat Anda gunakan, seperti kontribusi dan hanya lihat. Jika mereka tidak memenuhi kebutuhan Anda, Anda dapat membuat tingkat izin tambahan. Cara termudah adalah melacak izin jika Anda tidak mengedit atau menghapus tingkat default, dan jika Anda menambahkan orang ke grup dan menetapkan tingkat izin ke grup tersebut, bukan menetapkan tingkat izin ke masing-masing pengguna.
  
- [Memahami tingkat izin](https://go.microsoft.com/fwlink/?linkid=867071)
    
- [Cara membuat dan mengedit tingkat izin](https://go.microsoft.com/fwlink/?linkid=867072)
