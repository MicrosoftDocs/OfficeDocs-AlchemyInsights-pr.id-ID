[Sumber daya untuk melakukan migrasi aplikasi ke Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migration-resources) -sumber daya untuk membantu Anda melakukan migrasi Access dan autentikasi aplikasi ke Azure Active Directory (Azure AD).

[Memindahkan autentikasi aplikasi dari Active Directory Federation Services ke Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migrate-adfs-apps-to-azure) -panduan pengembang yang difokuskan untuk melakukan migrasi aplikasi dari Active Directory Federation Services ke Azure Active Directory.
