# <a name="cant-create-a-team-in-teams"></a>Tidak dapat membuat tim di teams

Admin teams Anda mungkin tidak memberikan izin yang diperlukan untuk membuat tim.  

Lihat admin tentang membuat atau mengubah pengaturan grup keamanan organisasi di Microsoft 365. Jika ditambahkan ke grup keamanan, Anda dapat membuat teams.
