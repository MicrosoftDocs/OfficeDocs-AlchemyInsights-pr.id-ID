# <a name="use-your-browser-to-check-spelling"></a>Gunakan browser Anda untuk memeriksa ejaan

Meskipun Outlook.com tidak memiliki fungsi spell check, banyak browser web secara otomatis memeriksa ejaan Anda, termasuk Microsoft Edge, Internet Explorer 10 dan versi yang lebih baru, dan versi Firefox, Chrome, dan Safari saat ini. Periksa pilihan untuk browser web Anda untuk mempelajari lebih lanjut tentang cara memeriksa ejaan.
