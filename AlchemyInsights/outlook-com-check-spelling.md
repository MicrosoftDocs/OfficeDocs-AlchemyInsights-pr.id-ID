# <a name="use-your-browser-to-check-spelling"></a>Menggunakan browser Anda untuk memeriksa ejaan

Meskipun Outlook.com tidak memiliki fungsi pemeriksa ejaan, banyak browser web secara otomatis memeriksa ejaan Anda, termasuk Microsoft Edge, Internet Explorer 10 dan versi yang lebih baru, serta versi terbaru Firefox, Chrome, dan Safari. Centang opsi untuk browser web Anda untuk mempelajari selengkapnya tentang cara memeriksa ejaan.
