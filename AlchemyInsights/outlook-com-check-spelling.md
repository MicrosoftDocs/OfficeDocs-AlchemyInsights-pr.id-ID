# <a name="use-your-browser-to-check-spelling"></a>Gunakan browser Anda untuk Periksa ejaan

Meskipun Outlook.com memiliki mantra memeriksa fungsi, banyak browser web secara otomatis memeriksa ejaan, termasuk Microsoft Edge, Internet Explorer 10 dan versi, dan versi terbaru Firefox, Chrome dan Safari. Periksa pilihan browser web Anda untuk mempelajari lebih lanjut tentang cara memeriksa ejaan.
