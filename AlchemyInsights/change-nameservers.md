# <a name="update-your-domain-nameservers-to-office-365"></a>Memperbarui nameserver domain Anda ke Office 365

Catatan: Perubahan Nameserver kadang-kadang dapat berlangsung hingga 48 jam untuk penggandaan.
  
Untuk mengatur domain Anda di Office 365, nameserver di registrar Anda perlu diperbarui. Membuat atau mengedit catatan nameserver Anda di domain registrar.
  
1. Pergi ke situs web domain registrar dan menemukan daerah dimana Anda bisa mengedit nameserver.
    
2. Membuat atau mengedit dua nameserver rekaman untuk mencocokkan nilai-nilai ini:
    
  - ns1.BDM.microsoftonline.com
    
  - ns2.BDM.microsoftonline.com
    
3. Simpan perubahan.
    
Anda juga dapat menemukan petunjuk rinci dalam artikel ini: [perubahan nameserver untuk mengatur Office 365 dengan setiap registrar domain](https://support.office.com/article/https://support.office.com/en-us/article/Change-nameservers-at-any-domain-registrar-to-set-up-Office-365-a8b487a9-2a45-4581-9dc4-5d28a47010a2.aspx)
