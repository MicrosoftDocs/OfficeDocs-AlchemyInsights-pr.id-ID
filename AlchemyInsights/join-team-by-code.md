# <a name="join-a-team-by-code"></a>Bergabung dalam tim dengan kode

Microsoft menyadari masalah ketika pengguna tidak dapat membuat **gabungan kode tim** , atau bergabung dalam tim menggunakan fitur **bergabung dengan kode** . Sebagai solusinya, pemilik tim dapat menambahkan pengguna secara manual, sebagaimana diperlukan.
