# <a name="how-to-share-sharepoint-files-or-folders"></a>Cara berbagi file atau folder SharePoint

File yang Anda simpan di situs SharePoint biasanya tersedia untuk semua orang dengan izin ke situs, namun Anda mungkin ingin berbagi file atau folder tertentu dengan orang yang tidak memiliki akses ke situs tersebut. Ketika Anda berbagi file dan folder, Anda dapat memutuskan apakah akan membiarkan orang mengedit atau hanya melihat mereka.
