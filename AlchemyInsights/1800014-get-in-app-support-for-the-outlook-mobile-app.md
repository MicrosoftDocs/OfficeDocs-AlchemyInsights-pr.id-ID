# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Dapatkan dukungan di-app untuk Outlook mobile app

1. Di Outlook mobile app, buka **Menu**.
    
2. Pilih tanda tanya untuk membuka **membantu &amp; umpan balik** layar. 
    
3. Tekan **Kontak dukungan** dan menjelaskan masalah Anda. 
    
4. Tekan **kirim** dan Anda akan menerima pemberitahuan di-app ketika agen dukungan kontak Anda. 
    
Untuk petunjuk rinci, lihat [Dapatkan bantuan di-app untuk Outlook untuk iOS dan Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
