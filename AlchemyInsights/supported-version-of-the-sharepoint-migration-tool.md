# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Versi yang didukung dari alat migrasi SharePoint



Saat ini alat migrasi SharePoint hanya mendukung migrasi dari SharePoint 2013. Versi 2010 dan 2016 tidak didukung saat ini.
  
Untuk informasi selengkapnya, lihat [memperkenalkan alat migrasi SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
