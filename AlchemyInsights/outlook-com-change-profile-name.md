# <a name="change-your-profile-name-in-outlookcom"></a>Mengubah nama profil Anda di Outlook.com

1. Pergi ke [Your Microsoft account info](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Pilih **Edit nama**.
3. Masukkan nama pertama dan nama terakhir, dan pilih **Simpan**.

Silakan tunggu hingga 24 jam untuk perubahan nama profil Anda untuk masuk ke efek.
