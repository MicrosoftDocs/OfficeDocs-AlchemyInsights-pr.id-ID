# <a name="change-your-profile-name-in-outlookcom"></a>Ubah nama profil Anda di Outlook.com

1. Buka [Info Akun Microsoft Anda](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Pilih **Edit nama**.
3. Masukkan nama depan dan nama belakang, lalu pilih **Simpan**.

Harap tunggu hingga 24 jam agar perubahan nama profil Anda mulai berlaku.
