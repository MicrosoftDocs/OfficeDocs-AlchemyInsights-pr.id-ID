# <a name="upload-files-to-onedrive-or-sharepoint"></a>Mengunggah file ke OneDrive atau SharePoint

Anda dapat mengunggah file ke pustaka dokumen di SharePoint online hanya dengan menyeretnya dari komputer dan menghapusnya ke pustaka dokumen. Dengan OneDrive, Anda dapat menyinkronkan file antara komputer dan awan agar dapat masuk ke file dari mana saja. Anda juga dapat bekerja dengan file yang disinkronkan secara langsung di file Explorer dan mengakses file Anda bahkan saat Anda offline. Untuk informasi selengkapnya tentang pengunggahan dan sinkronisasi, silakan kunjungi tautan di bawah ini.
