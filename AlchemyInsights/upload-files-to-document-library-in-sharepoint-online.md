# <a name="upload-files-to-onedrive-or-sharepoint"></a>Unggah file ke OneDrive atau SharePoint

Anda dapat mengunggah file ke pustaka dokumen di SharePoint online hanya dengan menyeretnya dari komputer Anda dan memasukkannya ke dalam pustaka dokumen. Dengan OneDrive, Anda dapat menyelaraskan file antara komputer dan Cloud, sehingga Anda bisa mendapatkan file Anda dari mana saja. Anda juga dapat bekerja dengan file Anda disinkronkan langsung di file Explorer dan mengakses file Anda bahkan ketika Anda sedang offline.
