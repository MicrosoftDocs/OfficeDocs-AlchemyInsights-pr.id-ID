# <a name="upload-files-to-onedrive-or-sharepoint"></a>Upload file ke OneDrive atau SharePoint

Anda dapat meng-upload file ke pustaka dokumen di SharePoint Online dengan hanya menyeret mereka dari komputer Anda dan menjatuhkan mereka ke dalam dokumen perpustakaan. Dengan OneDrive, Anda dapat menyinkronkan file antara komputer dan internet, sehingga Anda bisa mendapatkan ke file Anda dari mana saja. Anda juga dapat bekerja dengan file yang disinkronkan secara langsung dalam File Explorer dan mengakses file Anda bahkan ketika Anda sedang offline. Untuk informasi lebih lanjut tentang meng-upload dan sinkronisasi, silakan kunjungi link di bawah ini.
