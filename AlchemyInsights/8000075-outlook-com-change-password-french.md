# <a name="changer-votre-mot-de-passe-dans-outlookcom"></a>Changer votre mot de passe dans Outlook.com

Votre mot de passe Outlook.com est le même que votre mot de passe Compte Microsoft.

1. Accédez a [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) et sélectionnez **pengubah de mot le passe**.
2. Par mesure de sécurité, vous pouvez d'etre invité a votre confirmer identité a l'aide d'un kode de sécurité.
3. Entrez votre mot de passe actuel, votre nouveau mot de passe, aku sélectionnez **Enregistrer**.

Si vous avez oublié votre mot de passe, accédez a la [halaman de réinitialisation de mot de passe de compte Microsoft](https://go.microsoft.com/fwlink/p/?linkid=841909), aku suivez les mengundang.

Si vous pensez que votre Outlook.com été piraté, voir [Mon compte Outlook.com été piraté](https://support.office.com/fr-fr/article/mon-compte-outlook-com-a-été-piraté-35993ac5-ac2f-494e-aacb-5232dda453d8).
