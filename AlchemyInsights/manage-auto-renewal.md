# <a name="manage-recurring-billing"></a>Mengelola penagihan berulang

Anda dapat mengaktifkan atau menonaktifkan penagihan berulang di pusat admin Microsoft 365.
  
1. Di pusat admin, buka halaman **penagihan** \> **[produk](https://go.microsoft.com/fwlink/p/?linkid=842054)** .

2. Di bagian **penagihan**, di samping **pembaruan** atau **kedaluwarsa**, pilih **Edit**.

3. Di panel **setelan penagihan berulang** **, pilih Matikan untuk mematikannya** . Jika penagihan berulang sudah dimatikan, pilih **bulanan** atau **sekali** untuk mengaktifkannya.

4. Pilih **kirim perubahan**.

Untuk informasi selengkapnya, lihat [Mengaktifkan atau menonaktifkan tagihan berulang](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/renew-your-subscription#turn-recurring-billing-off-or-on).
