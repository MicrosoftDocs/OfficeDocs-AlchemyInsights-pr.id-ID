# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>Izin dan persyaratan yang terkait dengan browser untuk Microsoft 365 Defender

- Izin: untuk mengaktifkan Microsoft 365 Defender, Anda harus menjadi admin global atau admin keamanan di Azure Active Directory. Untuk melihat daftar peran yang bisa menggunakan Microsoft 365 Defender, dan untuk mempelajari bagaimana akses ke data diatur, lihat [mengelola akses ke microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2143626).

- Persyaratan terkait browser: untuk mengakses Microsoft 365 Defender di pusat keamanan Microsoft 365, gunakan Microsoft Edge, Internet Explorer 11, atau HTML 5-compliant web browser.
