# <a name="open-a-second-browser-window"></a>Membuka jendela browser kedua

Untuk membuka browser kedua, lakukan langkah-langkah berikut:

1. Di jendela browser pertama (tempat Anda membaca ini), pastikan bahwa panduan domain terbuka. Jika Anda tidak mengklik tombol di atas untuk memulai panduan domain, Anda bisa masuk ke panduan domain dengan masuk ke domain **> menambahkan domain**.

2. Buka jendela browser kedua. Di browser kedua, Anda akan mengikuti langkah berikutnya.
