# <a name="learn-the-language-for-creating-advanced-hunting-queries"></a>Mempelajari bahasa untuk membuat kueri berburu tingkat lanjut

Perburuan tingkat lanjut didasarkan pada [bahasa kueri Kusto](https://go.microsoft.com/fwlink/?linkid=2144620). Anda dapat menggunakan sintaks dan operator Kusto untuk menyusun kueri yang menemukan informasi dalam [skema](https://go.microsoft.com/fwlink/?linkid=2144621) yang secara khusus terstruktur untuk berburu tingkat lanjut.

Untuk mempelajari selengkapnya, lihat [mempelajari bahasa kueri berburu tingkat lanjut](https://go.microsoft.com/fwlink/?linkid=2144518).
