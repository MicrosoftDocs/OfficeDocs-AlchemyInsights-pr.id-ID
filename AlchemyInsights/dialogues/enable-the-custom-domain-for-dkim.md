# <a name="enable-the-custom-domain-for-dkim"></a>Mengaktifkan domain kustom untuk DKIM

Setelah Anda membuat catatan CNAME untuk domain kustom Anda, Anda perlu mengaktifkan domain tersebut.

Untuk mengaktifkan domain, lakukan langkah-langkah berikut:

1. Navigasikan ke [Pusat admin Exchange](https://outlook.office365.com/ecp/).
2. Di panel kiri, pilih **proteksi > DKIM**.
3. Pilih domain, lalu di bawah **tandatangani pesan untuk domain ini dengan tanda tangan DKIM**, klik **Aktifkan**. Ulangi langkah ini untuk setiap domain.
