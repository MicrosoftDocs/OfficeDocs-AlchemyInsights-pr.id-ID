# <a name="report-suspicious-urls"></a>Laporkan URL yang mencurigakan

Untuk mengirimkan URL mencurigakan, ikuti langkah berikut:

1. Dalam Flyout **penyerahan baru** , masukkan URL lengkap termasuk protokol (https://).
2. Jika Anda memilih **seharusnya diblokir**, Tentukan apakah URL tersebut adalah **Pengelabuan** atau **malware**.
3. Pilih **kirim**.
