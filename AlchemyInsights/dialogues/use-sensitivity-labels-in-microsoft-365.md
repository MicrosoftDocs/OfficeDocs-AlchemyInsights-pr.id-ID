# <a name="use-sensitivity-labels-in-microsoft-365"></a>Menggunakan label sensitivitas di Microsoft 365

Anda dapat menggunakan label sensitivitas untuk:
- Terapkan pengaturan proteksi seperti enkripsi atau cap air pada konten berlabel.
- Lindungi konten di aplikasi Office di berbagai platform dan perangkat.
- Lindungi konten dalam aplikasi dan layanan pihak ketiga dengan menggunakan keamanan Microsoft Cloud App.
- Memperluas label sensitivitas ke aplikasi dan layanan pihak ketiga.
- Mengklasifikasikan konten tanpa menggunakan pengaturan proteksi apa pun.

Untuk informasi selengkapnya, lihat [mempelajari tentang label sensitivitas](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels).
