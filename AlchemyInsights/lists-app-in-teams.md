# <a name="lists-app-in-teams"></a>Daftar aplikasi di teams

Aplikasi daftar di Microsoft teams membawa daftar di situs tim SharePoint ke dalam kolaborasi dan kanvas tim kerja saluran. Dengan aplikasi ini, pengguna akhir dapat mengerjakan daftar favoritnya dari dalam teams.  

Fitur ini mengirim default ke aktif. Daftar telah diinstal sebelumnya untuk semua pengguna teams dan tersedia langsung di Galeri tab setiap tim dan saluran.  

Admin teams dapat mengontrol ketersediaan aplikasi menggunakan kebijakan izin Microsoft teams. Untuk informasi selengkapnya, lihat [mengelola kebijakan izin aplikasi di Microsoft teams](https://docs.microsoft.com/microsoftteams/teams-app-permission-policies).

**Pengaturan waktu:** Awal September dan lengkap pada pertengahan Oktober 2020  

**Peluncuran:** Tingkat organisasi  

**Tipe kontrol:**  Kontrol admin teams  

**Tindakan:**  Meninjau dan menilai
