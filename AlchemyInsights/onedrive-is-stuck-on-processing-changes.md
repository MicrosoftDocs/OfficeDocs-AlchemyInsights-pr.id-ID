# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive terhenti saat memproses perubahan

Jika OneDrive memperlihatkan "memproses perubahan" untuk waktu yang lama, mungkin karena Anda memiliki file online yang terbuka, banyak file terjebak dalam antrian, atau file yang sangat besar yang disinkronkan. Baca  [OneDrive terhenti pada "memproses perubahan"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) untuk langkah yang disarankan dan informasi selengkapnya.

Jika Anda memerlukan bantuan lainnya, Anda bisa  [mengirimkan email dukungan OneDrive](https://go.microsoft.com/fwlink/p/?LinkId=528676).
