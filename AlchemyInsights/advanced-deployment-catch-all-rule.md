# <a name="deploy-office-apps"></a>Menyebarkan aplikasi Office

Untuk menyebarkan aplikasi Office untuk beberapa pengguna akhir melalui alat penyebaran Office, Microsoft Endpoint Configuration Manager, sebagai bagian dari gambar disk sistem operasi, atau dalam skenario komputer bersama menggunakan Windows Server dengan layanan desktop jarak jauh (RDS) atau layanan terminal, pertimbangkan untuk menggunakan panduan langkah demi langkah di **Office 365 ProPlus operator penyebaran**.
  
Untuk memulai, pastikan bahwa Anda masuk sebagai admin penghuni atau global admin, dan kemudian klik di sini: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
