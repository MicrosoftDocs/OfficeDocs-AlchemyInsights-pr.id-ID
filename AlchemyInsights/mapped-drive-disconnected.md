# <a name="sharepoint-libraries-mapped-to-network-drives-become-disconnected"></a>Pustaka SharePoint yang dipetakan ke drive jaringan terputus

Pemetaan pustaka dokumen SharePoint ke drive jaringan bersifat sementara. Anda sesekali harus membuka situs SharePoint di Internet Explorer dan pilih **tetap masuk** untuk mencegah sesi kedaluwarsa. 
  
[Dapatkan bantuan mengonfigurasi dan memecahkan masalah drive jaringan yang dipetakan](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives).
  
Untuk bekerja dengan pustaka di file Explorer, [sinkronkan file SharePoint dengan klien sinkronisasi OneDrive baru](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) yang menyertakan [file sesuai permintaan](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Mengakses semua file Anda di OneDrive tanpa menggunakan ruang penyimpanan lokal.
