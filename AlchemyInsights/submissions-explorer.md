# <a name="new-feature-to-help-diagnose-eop-and-atp"></a>Fitur baru untuk membantu mendiagnosa EOP dan ATP

Sebagai admin, Anda dapat sekarang dengan mudah melaporkan email mencurigakan pesan, URL, dan konten ke Microsoft di [Explorer pengiriman](https://protection.office.com/reportsubmission). Anda bahkan dapat menggunakannya untuk membantu Anda memahami mengapa pesan sah ditandai salah. Untuk informasi lebih lanjut, lihat ini [pengumuman](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).
