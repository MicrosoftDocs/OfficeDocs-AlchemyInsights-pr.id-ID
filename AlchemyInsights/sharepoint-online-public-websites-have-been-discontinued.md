# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>SharePoint Online situs web publik telah dihentikan

SharePoint Online situs web publik telah dihentikan.

Jika Anda membuat situs web umum SharePoint Online, Anda mampu untuk terus menggunakannya melalui Maret 31, 2018. Jika Anda ingin untuk terus memiliki situs web umum, Anda perlu membuat akun baru dengan penyedia baru. Konten di situs umum SharePoint Online telah dihapus.

Untuk info tentang penghentian situs, lihat [rincian tentang proses penghentian fitur](https://go.microsoft.com/fwlink/?linkid=866980).
