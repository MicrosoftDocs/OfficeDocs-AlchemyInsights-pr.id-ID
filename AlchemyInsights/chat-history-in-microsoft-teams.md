# <a name="viewing-chat-history-in-microsoft-teams"></a>Melihat riwayat chat di Microsoft teams

Untuk melihat riwayat obrolan tim, Anda dapat menggunakan [pencarian konten](https://sip.protection.office.com/contentsearchbeta?ContentOnly=1) dari [pusat kepatuhan & keamanan](https://sip.protection.office.com/insightdashboard), di mana Anda dapat mencari kotak pesan yang terkait dengan Microsoft 365 grup atau Microsoft Team. 

Lihat [menelusuri Microsoft teams dan microsoft 365 grup](https://docs.microsoft.com/microsoft-365/compliance/content-search) untuk mempelajari tentang kemampuan penelusuran dari pusat kepatuhan dan keamanan. 

**Catatan:** Mencadangkan, mengekspor, mencetak, dan mengarsipkan Riwayat percakapan saat ini tidak tersedia langsung dari Microsoft teams. Silakan memilih kemampuan ini di [Uservoice](https://microsoftteams.uservoice.com/forums/555103-public/suggestions/16982542-backup-export-printing-archive-options?page=2&per_page=20).
