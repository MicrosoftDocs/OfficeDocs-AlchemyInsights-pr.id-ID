# <a name="site-and-content-structure-removed"></a>Struktur situs dan konten dihapus

Mulai bulan Oktober 2018, halaman konten situs dan struktur akan tidak lagi didukung atau menerima pembaruan fitur. Pengaturan link ke halaman SiteManager. aspx akan dihapus dari SharePoint online dan akses langsung akan dibatasi untuk situs koleksi admin hingga 2019 Maret, pada saat halaman konten situs dan struktur akan sepenuhnya dihentikan. 

Catatan: sebagai penyelesaian sampai fitur sepenuhnya dihentikan, administrator koleksi situs dapat menavigasi ke Microsoft 365 Admin Center, pilih ubin SharePoint, dan menambahkan atau mengedit URL dengan mengganti '/SharePoint.aspx ' dengan '/Sitemanager.aspx '. 


Gunakan Pindahkan dan salin file untuk mengelola konten dan struktur situs. Untuk informasi selengkapnya, lihat [menyalin berkas dan map antara OneDrive dan SharePoint situs](https://support.office.com/article/copy-files-and-folders-between-onedrive-and-sharepoint-sites-67a6323e-7fd4-4254-99a8-35613492a82f). 

Untuk informasi lebih lanjut, silakan merujuk ke [bekerja dengan situs konten dan struktur](https://support.office.com/article/Work-with-site-content-and-structure-30fcaad9-02b1-4347-8b03-e1ccc5a4c19f).
