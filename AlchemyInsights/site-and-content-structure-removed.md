# <a name="site-and-content-structure-removed"></a>Situs dan konten struktur dihapus

Dimulai pada Oktober tahun 2018, situs konten dan halaman struktur akan tidak lagi didukung atau menerima pembaruan fitur. Pengaturan link ke halaman SiteManager.aspx akan dihapus dari SharePoint Online dan langsung akses akan dibatasi untuk admin situs koleksi hingga 2019 Maret, saat halaman konten situs dan struktur akan sepenuhnya pensiun. 

Catatan: Sebagai solusi sampai fitur sepenuhnya pensiunan, Site Koleksi Administrator dapat menavigasi ke Portal 365 kantor-> klik pada SharePoint ubin - > menambahkan atau mengedit URL dengan mengganti ' / Sharepoint.aspx' dengan ' / Sitemanager.aspx'. 


Gunakan File bergerak dan salinan untuk mengelola konten situs dan struktur. Untuk informasi lebih lanjut, lihat [menyalin file dan folder antara OneDrive dan SharePoint situs](https://support.office.com/article/copy-files-and-folders-between-onedrive-and-sharepoint-sites-67a6323e-7fd4-4254-99a8-35613492a82f). 

Untuk informasi lebih lanjut, silakan lihat untuk [bekerja dengan konten situs dan struktur](https://support.office.com/article/Work-with-site-content-and-structure-30fcaad9-02b1-4347-8b03-e1ccc5a4c19f).
