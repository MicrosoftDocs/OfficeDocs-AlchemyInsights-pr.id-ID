# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>Masalah bebas/sibuk kalender di lingkungan hibrid

Jika pengguna mengalami masalah melihat informasi bebas/sibuk kalender di lingkungan hibrid, jalankan kembali Wisaya konfigurasi hibrid. Untuk informasi selengkapnya tentang Wisaya konfigurasi hibrid, lihat [Wisaya konfigurasi hibrid](https://go.microsoft.com/fwlink/p/?linkid=528149).
