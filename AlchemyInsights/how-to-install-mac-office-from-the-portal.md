# <a name="how-to-install-mac-office-from-the-portal"></a>Bagaimana menginstal Mac kantor dari portal

1. [Uninstall Office 2011 untuk Mac dari komputer Anda](https://support.office.com/article/4bfcd230-0ea1-4656-bf30-dbfa44d358fa?wt.mc_id=Alchemy_ClientDIA).

2. [Download perangkat lunak memastikan Mac diaktifkan pada admin portal Office 365](https://support.office.com/article/c13051e6-f75c-4737-bc0d-7685dcedf360?wt.mc_id=Alchemy_ClientDIA).

3. [Menetapkan lisensi Office untuk pengguna Mac Anda](https://support.office.com/article/997596B5-4173-4627-B915-36ABAC6786DC?wt.mc_id=Alchemy_ClientDIA).

4. Pengguna dapat kemudian instal perangkat lunak perkantoran sendiri, atau Anda dapat menggunakan [Menyebarkan kantor untuk pengguna Mac Anda](https://docs.microsoft.com/DeployOffice/mac/deployment-guide-for-office-for-mac).
