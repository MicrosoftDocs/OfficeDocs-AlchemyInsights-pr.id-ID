# <a name="put-apps-on-the-desktop"></a>Menempatkan aplikasi pada desktop

Cari aplikasi di menu **Start** , dan ambil ikon aplikasi dengan menekan tombol kiri mouse dan memindahkan ikon ke desktop.

Anda juga dapat menekan **tombol Windows + R** dan kemudian ketik perintah ini:

`shell:appsfolder`

Ini akan membuka folder **aplikasi** di **file Explorer**. Anda sekarang dapat menyeret dan menjatuhkan ikon aplikasi ke desktop.
