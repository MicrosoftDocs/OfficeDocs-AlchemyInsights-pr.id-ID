# <a name="put-apps-on-the-desktop"></a>Meletakkan aplikasi di desktop

Cari aplikasi di menu **mulai** , dan ambil ikon aplikasi dengan menahan tombol mouse kiri dan memindahkan ikon ke desktop.

Anda juga dapat menekan **tombol Windows + R** lalu ketikkan perintah ini:

`shell:appsfolder`

Ini akan membuka folder **aplikasi** di **file Explorer**. Kini Anda dapat menyeret dan menjatuhkan ikon aplikasi ke desktop.
