# <a name="change-your-pin"></a>Mengubah PIN Anda

Outlook.com tidak menggunakan PIN, tetapi Anda mungkin masuk ke PC Anda dengan PIN atau Windows Hello. Untuk mengubah PIN Anda di Windows 10:

1. Pilih tombol mulai atau tekan tombol Windows di keyboard Anda, lalu pilih **setelan** > **akun** > **masuk opsi**.
2. Pilih tombol **Ubah** di bawah **pin**.
3. Masukkan PIN Anda saat ini, lalu masukkan dan konfirmasikan PIN baru Anda di bawahnya.

[Baca lebih lanjut tentang Windows Hello.](https://support.microsoft.com/help/17215/)
