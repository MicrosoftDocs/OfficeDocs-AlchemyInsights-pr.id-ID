# <a name="change-your-pin"></a>Mengubah PIN

Outlook.com tidak menggunakan PIN, tapi Anda mungkin masuk ke PC Anda dengan PIN atau Windows Halo. Untuk mengubah PIN di Windows 10:

1. Pilih tombol Start atau tekan tombol Windows di keyboard Anda, kemudian pilih **pengaturan** > **account** > **masuk pilihan**.
2. Pilih tombol **Ubah** di bawah **PIN**.
3. Masukkan PIN aktif, kemudian masukkan dan konfirmasikan PIN baru di bawah.

[Baca lebih lanjut tentang Windows Halo.](https://support.microsoft.com/help/17215/)
