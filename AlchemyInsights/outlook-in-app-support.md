# <a name="in-app-support-in-outlook-for-windows"></a>Dukungan dalam aplikasi di Outlook untuk Windows

Sebagai admin penghuni, Anda dapat menerima dukungan dalam aplikasi untuk masalah di Outlook untuk Windows. Di Outlook, pilih **bantuan**  >  **Hubungi dukungan**. Masukkan pertanyaan Anda, dan anggota tim dukungan Outlook untuk Windows akan membalas pesan Anda.
