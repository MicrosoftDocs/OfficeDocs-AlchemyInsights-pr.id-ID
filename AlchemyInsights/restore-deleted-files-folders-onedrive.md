# <a name="restore-deleted-files-or-folders-in-onedrive"></a>Kembalikan file atau folder yang dihapus di OneDrive

Jika Anda tidak sengaja menghapus berkas atau map di OneDrive, Anda mungkin dapat memulihkan kemudian dari OneDrive recycle bin.

* [Kembalikan file atau folder yang dihapus di OneDrive](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)

Jika banyak file OneDrive Anda dihapus, ditimpa, rusak, atau terinfeksi oleh malware, Anda dapat memulihkan seluruh OneDrive Anda ke waktu sebelumnya.

* [Memulihkan OneDrive Anda](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
