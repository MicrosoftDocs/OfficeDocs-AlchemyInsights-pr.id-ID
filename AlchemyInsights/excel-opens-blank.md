Jika Excel dimulai tapi menampilkan layar kosong tempat Anda mengharapkan file muncul, lihat [memecahkan masalah pembukaan Excel kosong saat Anda mengklik ganda ikon file atau nama file](https://docs.microsoft.com/office/troubleshoot/excel/excel-opens-blank).
