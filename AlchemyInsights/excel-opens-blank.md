Jika Excel dimulai tetapi menampilkan layar kosong di mana Anda harapkan file untuk muncul, lihat [mengatasi masalah Excel membuka kosong ketika Anda klik dua kali ikon file atau nama file](https://docs.microsoft.com/office/troubleshoot/excel/excel-opens-blank).
