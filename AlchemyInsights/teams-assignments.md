# <a name="teams-assignments"></a>Tugas tim

Penugasan adalah tugas atau unit pekerjaan yang ditugaskan kepada seorang siswa atau anggota tim di kelas sebagai bagian dari penelaahan mereka. Anda dapat membuat tugas dalam kelas teams.

- Untuk mempelajari lebih lanjut tentang tugas, lihat [penugasan Microsoft teams](https://support.microsoft.com/en-us/office/microsoft-teams-5aa4431a-8a3c-4aa5-87a6-b6401abea114#ID0EAABAAA=Assignments).
- Untuk mempelajari lebih lanjut tentang mengelola fitur tugas di pusat admin Microsoft teams, lihat [tugas di tim untuk pendidikan](https://docs.microsoft.com/microsoftteams/expand-teams-across-your-org/assignments-in-teams).
