# <a name="intune-and-device-actions"></a>Intune dan tindakan perangkat

Intune memberi admin kemampuan untuk menjalankan tindakan perangkat dari jarak jauh. Dengan menggunakan tindakan ini dari portal Intune, Anda dapat memulai ulang perangkat, mereset kode akses, menemukan perangkat yang hilang atau dicuri, dan banyak lagi. Tinjau [portal tindakan perangkat](https://docs.microsoft.com/mem/intune/remote-actions/) untuk mempelajari selengkapnya.
