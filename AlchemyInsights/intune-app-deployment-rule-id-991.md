# <a name="intune-app-deployment"></a>Penggunaan aplikasi Intune

Menyebarkan aplikasi dengan Intune adalah proses tiga langkah:
  
1. **Menambahkan aplikasi** -Anda bisa menambahkan aplikasi App Store atau mengunggah baris aplikasi bisnis.

2. **Menetapkan aplikasi** -menentukan lingkup penyebaran aplikasi.

3. **Aplikasi pemantauan** -menampilkan status penginstalan aplikasi, daftar ditemukan aplikasi, dan mengelola aplikasi yang dibeli dari toko aplikasi.

Baca [apa itu manajemen aplikasi Intune](https://docs.microsoft.com/intune/app-management) untuk panduan dan informasi tentang mengonfigurasi dan memproteksi aplikasi.
