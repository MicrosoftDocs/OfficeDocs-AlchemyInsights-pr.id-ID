# <a name="how-to-access-outlookcom-while-traveling"></a>Cara mengakses Outlook.com saat bepergian

Untuk mengakses Outlook.com saat bepergian, pastikan Info keamanan akun Microsoft Anda-alamat email atau nomor telepon alternatif tempat Anda dapat memperoleh kode keamanan-adalah yang terbaru. Untuk mempelajari lebih lanjut, lihat [cara mengakses Outlook.com saat bepergian](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
