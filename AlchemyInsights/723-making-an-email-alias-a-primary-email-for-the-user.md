# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>Membuat email alias alamat primer untuk pengguna

Untuk petunjuk, lihat [menambahkan alias email lain untuk pengguna](https://support.office.com/article/0b0bd900-68b1-4bf5-808b-5d240a7739f4).
  
 **Catatan**: jika organisasi Anda dalam Hybrid mode, Anda perlu mengubah alamat email utama di organisasi Exchange lokal. Untuk selengkapnya, lihat [Alamat email Tambah atau Hapus untuk kotak pesan](https://technet.microsoft.com/library/bb123794.aspx).
