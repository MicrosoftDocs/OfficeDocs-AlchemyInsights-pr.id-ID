# <a name="create-and-use-a-shared-mailbox"></a>Membuat dan menggunakan kotak pesan bersama

Kotak pesan bersama memungkinkan lebih dari satu orang mengirim dan menerima email untuk alias, seperti **sales@contoso.com**, tanpa menentukan lisensi lain.
  
[Buat kotak pesan bersama](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) menjelaskan bagaimana: 
  
- Membuat kotak surat bersama
    
- Memberikan pengguna izin untuk menggunakan kotak pesan bersama
    
- Mengakses kotak pesan bersama di Outlook atau Outlook di web
