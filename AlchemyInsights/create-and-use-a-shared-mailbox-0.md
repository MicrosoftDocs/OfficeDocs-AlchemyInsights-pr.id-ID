# <a name="create-and-use-a-shared-mailbox"></a>Membuat dan menggunakan kotak surat bersama

Kotak surat bersama memungkinkan lebih dari satu orang mengirim dan menerima email untuk alias, seperti **Sales@contoso.com**, tanpa menetapkan lisensi lain.
  
[Membuat kotak pesan bersama](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) menjelaskan cara: 
  
- Membuat kotak surat bersama
    
- Memberikan izin kepada pengguna untuk menggunakan kotak pesan bersama
    
- Mengakses kotak surat bersama di Outlook atau Outlook di web
