# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Menggunakan kotak pesan bersama di perangkat seluler

Anda tidak dapat mengakses kotak surat bersama dari aplikasi Outlook yang diinstal di perangkat seluler. Apakah Anda ingin fitur ini? Biarkan tim Outlook Engineering tahu. [Suara di sini](https://go.microsoft.com/fwlink/?linked=862116)!
  
Anda hanya dapat menggunakan kotak surat bersama di perangkat seluler dengan membuka browser, masuk ke Office 365, dan kemudian pergi ke Outlook di web. Untuk petunjuk, lihat [menambahkan kotak surat bersama di Outlook di web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
  
Jika Anda harus mengakses kotak surat bersama menggunakan aplikasi Outlook pada perangkat mobile, pertimbangkan untuk membuat Office 365 grup alih-alih menggunakan kotak surat bersama. Untuk selengkapnya, lihat [membandingkan grup](https://docs.microsoft.com/office365/admin/create-groups/compare-groups).
  
Pelajari lebih lanjut: [Dapatkan bantuan dalam aplikasi untuk Outlook untuk IOS dan Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
