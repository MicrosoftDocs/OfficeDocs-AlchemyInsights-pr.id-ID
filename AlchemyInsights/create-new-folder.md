# <a name="create-a-new-folder-in-outlookcom"></a>Buat folder baru di Outlook.com

1. Di bawah **folder** di panel navigasi, pilih **folder baru**.

2. Ketik nama untuk folder dan tekan Enter.

Untuk mempelajari lebih lanjut, lihat [membuat folder di Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
