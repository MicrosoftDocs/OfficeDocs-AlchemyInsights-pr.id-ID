# <a name="cant-edit-the-notes-field-for-a-contact-in-outlook"></a>Tidak dapat mengedit bidang catatan untuk kontak di Outlook
Ini adalah masalah umum. Kami sedang berusaha memperbaikinya dan status akan diperbarui dalam [tidak dapat mengedit bidang catatan untuk kontak](https://support.office.com/article/fb8394ce-04ce-48b5-bae4-be46f77f10fe).

Penanganan **masalah**: jika Anda menggunakan akun Microsoft 365 atau Outlook.com, Anda bisa mengedit kontak dengan menggunakan web. Jika Anda menggunakan akun IMAP atau POP, solusi tidak tersedia saat ini.
