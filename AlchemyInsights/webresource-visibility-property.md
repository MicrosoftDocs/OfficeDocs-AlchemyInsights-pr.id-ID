# <a name="webresource-visibility-property"></a>Properti visibilitas webresource

Anda harus memilih opsi yang **terlihat secara default** untuk sumber daya web muncul di formulir dan tersedia bagi pengguna. Untuk info lebih lanjut, lihat [properti sumber daya web untuk formulir aplikasi berdasarkan model](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy).
