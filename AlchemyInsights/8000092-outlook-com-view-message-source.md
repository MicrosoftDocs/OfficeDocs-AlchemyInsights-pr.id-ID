# <a name="how-to-view-message-headers-or-message-source"></a>Cara melihat header pesan atau pesan sumber

Untuk melihat header dan rincian lainnya dalam email, pilih tiga titik-titik di sudut atas panel baca dan pilih **Lihat pesan sumber**.
