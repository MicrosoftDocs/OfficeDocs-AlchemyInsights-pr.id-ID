# <a name="using-office-product-keys"></a>Menggunakan kunci produk Office

**Kunci produk Office**

Gunakan akun Microsoft Anda, bukan kunci produk, untuk menginstal dan mengaktifkan Office dan secara individual membeli aplikasi Microsoft 365, seperti Project, Visio, Word, Excel, dan Outlook. Untuk informasi selengkapnya, lihat [Office meminta saya untuk kunci produk](https://support.office.com/article/12a5763a-d45c-4685-8c95-a44500213759?ui=en-US&rs=en-US&ad=US#bkmk_promptforpkey)

- Jika pembelian Office datang dengan kunci produk, lihat [tempat untuk memasukkan kunci produk Office Anda](https://support.office.com/article/Where-to-enter-your-Office-product-key-0a82e5ae-739e-4b92-a6f4-2ec780c185db).

- Untuk mendapatkan bantuan dengan kunci produk Office 365 Home atau Personal, lihat [menggunakan kunci produk dengan Office](https://support.office.com/article/using-product-keys-with-office-12a5763a-d45c-4685-8c95-a44500213759).

**Mengaktifkan langganan Office** 

- Jika Anda membeli langganan Microsoft 365 dari toko ritel, lihat [memasukkan kunci produk Anda untuk aplikasi microsoft 365 untuk bisnis](https://docs.microsoft.com/microsoft-365/commerce/enter-your-product-key).

- Jika ada pesan kesalahan ketika Anda mencoba menebus kunci produk, lihat [masalah dengan kunci produk Microsoft 365 untuk bisnis Anda?](https://docs.microsoft.com/microsoft-365/commerce/product-key-errors-and-solutions)
