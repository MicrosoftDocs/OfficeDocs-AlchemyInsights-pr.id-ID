 <span data-ttu-id="4cbe0-106">**My antivirus menyebabkan mereka untuk membuka hanya-baca**</span><span class="sxs-lookup"><span data-stu-id="4cbe0-106">**My antivirus is causing them to open read-only**</span></span>

BitDefender, misalnya, memiliki kandungan menambahkan aplikasi pengecualian di sini: [Bagaimana menambahkan aplikasi atau proses pengecualian di Bitdefender kontrol pusat](https://www.bitdefender.com/support/how-to-add-application-or-process-exclusions-in-bitdefender-control-center-1119.mdl).</span><span class="sxs-lookup"><span data-stu-id="4cbe0-p102">Some antivirus programs may protect you from potentially unsafe files by opening them read-only.  <span data-ttu-id="4cbe0-110">**File properti diatur untuk hanya-baca?**</span><span class="sxs-lookup"><span data-stu-id="4cbe0-110">**Are the file properties set to read-only?**</span></span>

 <span data-ttu-id="4cbe0-113">**Konten yang dilindungi pemandangan**</span><span class="sxs-lookup"><span data-stu-id="4cbe0-113">**The content is in protected view**</span></span>

By using Protected View, you can read a file and see its contents while reducing the risks.
