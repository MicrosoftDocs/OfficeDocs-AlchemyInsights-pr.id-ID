# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Mengatur penerusan ke beberapa alamat email

Jika Anda perlu untuk mengatur email forwarding untuk kelompok pengguna, membuat daftar distribusi, menambahkan beberapa pengguna untuk itu dan kemudian mengkonfigurasi penerusan ke daftar itu. Pertama, arahkan ke [kelola grup](https://portal.office.com/adminportal/home#/groups)
  
1. Membuat *Daftar distribusi*

2. Menambahkan beberapa pengguna untuk itu

3. Kemudian mengkonfigurasi penerusan pesan ke grup
