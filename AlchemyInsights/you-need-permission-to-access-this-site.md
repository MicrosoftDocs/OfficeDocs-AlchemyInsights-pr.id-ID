# <a name="you-need-permission-to-access-this-site"></a>Anda perlu izin untuk mengakses situs ini

Ada banyak skenario yang dapat meminta salah satu pesan ini. Paling sering, izin untuk pengguna atau administrator dikonfigurasi dengan benar atau tidak dikonfigurasi sama sekali. 

Untuk informasi selengkapnya, baca "[Akses ditolak," "Anda memerlukan izin untuk mengakses situs ini," atau "pengguna tidak ditemukan di direktori" kesalahan dalam SharePoint online dan OneDrive untuk bisnis](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business).
