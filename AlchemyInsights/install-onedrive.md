# <a name="install-onedrive"></a>Menginstal OneDrive

OneDrive sync klien yang terinstal dengan Windows 10. Untuk menginstal OneDrive untuk Windows 7, Mac, atau Linux, atau untuk mendapatkan aplikasi mobile untuk ponsel atau tablet Anda, pergi ke [http://onedrive.com/download](http://onedrive.com/download).
