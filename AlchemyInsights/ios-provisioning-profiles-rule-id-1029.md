# <a name="ios-provisioning-profiles"></a>iOS Provisioning profil

Apple iOS line-of-business menggunakan konsep Profil bawaan yang digunakan untuk memvalidasi integritas dari aplikasi dan menerapkan kebijakan yang ditetapkan dalam profil. Ini penyediaan profil berakhir setelah 12 bulan dan jadi profil baru harus dikaitkan dengan app digunakan agar aplikasi untuk terus berjalan.
  
Artikel berikut menjelaskan cara membuat asosiasi baru penyediaan profil dengan iOS LOB aplikasi disebarkan melalui Intune: [penggunaan iOS aplikasi bawaan profil untuk mencegah aplikasi dari kedaluwarsa](https://docs.microsoft.com/intune/app-provisioning-profile-ios)
