# <a name="fix-problems-uploading-files-to-onedrive"></a>Memperbaiki masalah mengunggah file ke OneDrive

Coba ini jika file tidak dapat diunggah:
  
- Ganti nama file dan coba lagi. Beberapa nama file dan karakter dalam nama file tidak diperbolehkan. 
    
- Jika Anda sedang menyinkronkan file, Pindahkan file dari folder OneDrive Anda (misalnya, pindahkan ke desktop Anda), dan saat status sinkronisasi OneDrive muncul sebagai "diperbarui," Pindahkan kembali.
