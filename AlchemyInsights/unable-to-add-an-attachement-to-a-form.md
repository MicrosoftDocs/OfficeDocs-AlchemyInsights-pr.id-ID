# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Tidak dapat menambahkan lampiran ke bentuk yang menghadap eksternal

Jika Anda mencoba untuk menambahkan lampiran ke survei Microsoft forms eksternal, fitur yang belum tersedia. Untuk melacak status permintaan fitur tersebut, buka [entri UserVoice ini](https://go.microsoft.com/fwlink/?linkid=2133069), dan tambahkan suara Anda.
