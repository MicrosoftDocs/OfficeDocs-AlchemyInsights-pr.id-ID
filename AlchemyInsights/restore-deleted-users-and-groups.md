# <a name="restore-deleted-users-and-groups"></a>Memulihkan pengguna dan grup yang dihapus

Untuk memulihkan pengguna yang dihapus secara tidak sengaja, lihat [memulihkan atau menghapus pengguna yang baru dihapus menggunakan Azure Active Directory](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-users-restore?context=azure/active-directory/users-groups-roles/context/ugr-context).

Untuk memulihkan grup yang dihapus, lihat [memulihkan grup Microsoft 365 yang dihapus di Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/groups-restore-deleted).
