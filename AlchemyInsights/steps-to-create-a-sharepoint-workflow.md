# <a name="steps-to-create-a-sharepoint-workflow"></a>Langkah-langkah untuk membuat alur kerja SharePoint

- Jika Anda dapat menambahkan SharePoint 2010 persetujuan alur kerja, lihat [tidak dapat menambahkan default penyetelan 2010](https://docs.microsoft.com/alchemyinsights/can-t-add-default-2010-approval-workflow).
- Cara membuat alur kerja:
    - [2013 alur kerja dari SharePoint desainer](https://docs.microsoft.com/sharepoint/dev/general-development/creating-a-workflow-by-using-sharepoint-designer-and-the-sharepoint-wo)
    - [Alur kerja 2010 dari SharePoint desainer](https://support.office.com/article/introduction-to-designing-and-customizing-workflows-32c9c0bf-5e20-4f74-8b9c-d3ea79f2962b)
    - [Pendahuluan untuk SharePoint alur kerja](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3)

- Alur kerja pemecahan masalah dan Debugging:
    - Debugging Workflow menyerupai versi sebelumnya.  Untuk informasi lebih lanjut, lihat [Debugging SharePoint alur kerja](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).
    - Untuk informasi selengkapnya mengenai pemecahan masalah, lihat [mengatasi SharePoint Server kesalahan validasi alur kerja](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio).
