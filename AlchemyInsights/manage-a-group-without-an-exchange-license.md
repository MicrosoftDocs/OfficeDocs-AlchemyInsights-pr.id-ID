# <a name="manage-a-group-without-an-exchange-license"></a>Mengelola grup tanpa lisensi Exchange

Jika akun admin Office 365 tidak memiliki kotak surat, Anda mungkin mendapatkan galat tentang lisensi produk hilang ketika Anda mencoba untuk mengedit Office 365 grup.
  
Karena Anda admin penyewa, itu mungkin bagi Anda untuk mengedit grup meskipun Anda tidak memiliki lisensi Exchange. Hanya pergi ke [Exchange Admin Center](https://outlook.office365.com/ecp.aspx) \> **grup** untuk mengelola Office 365 grup.
  
Jika Anda ingin mengedit grup di pusat admin, tambahkan lisensi Exchange ke akun admin Anda. Anda harus menunggu hingga kotak pesan dibuat sepenuhnya agar dapat mengedit grup.
