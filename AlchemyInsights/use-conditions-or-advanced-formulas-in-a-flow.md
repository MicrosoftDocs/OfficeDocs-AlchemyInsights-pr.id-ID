# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Gunakan kondisi atau formula canggih dalam aliran

Di Microsoft Flow, Anda dapat menyesuaikan aliran sehingga melakukan tugas hanya jika kondisi yang benar. Untuk info, lihat [menambahkan kondisi untuk sebuah aliran](https://go.microsoft.com/fwlink/?linkid=872112).
  
Untuk membuat formula canggih, lihat [definisi Workflow skema bahasa untuk aplikasi Azure logika](https://aka.ms/logicexpressions).
