# <a name="setting-up-visio"></a>Menyiapkan Visio

Sepertinya Anda memerlukan bantuan untuk menyiapkan Visio dengan langganan Microsoft 365 Anda.
  
Pemberitahuan untuk pengguna Mac: produk Visio tidak tersedia untuk MacOS saat ini. Jika memiliki PC, Anda dapat menginstal perangkat lunak menggunakan langkah-langkah di bawah ini.
  
Anda dapat mengikuti langkah-langkah dalam [artikel ini](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) untuk menyiapkan penyiapan.
