# <a name="how-to-select-all-in-outlookcom"></a>Bagaimana "Pilih Semua" di Outlook.com

Anda hanya dapat memilih semua jika semua pesan terlihat. Hal ini untuk mencegah kecelakaan tindakan pada pesan Anda tidak dapat melihat.

**CTRL + klik** memilih beberapa pesan individu.

**SHIFT + klik** memilih semua pesan antara pesan pertama dan terakhir yang Anda pilih.
