# <a name="my-status-always-shows-as-available"></a>Status saya selalu diperlihatkan sebagai tersedia

Setiap kali Anda masuk ke Outlook.com, status akun Anda akan diatur ulang ke tersedia. Anda bisa mencegah status Anda berubah dengan tidak keluar dari Outlook.com. Untuk tetap masuk saat menutup browser Anda, pilih kotak **biarkan saya tetap masuk** di halaman masuk Outlook.com.
