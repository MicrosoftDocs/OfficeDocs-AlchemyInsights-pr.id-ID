# <a name="my-status-always-shows-as-available"></a>Status saya selalu menunjukkan sebagai tersedia

Setiap kali Anda masuk ke Outlook.com, status akun Anda akan disetel ulang ke tersedia. Anda dapat mencegah status Anda berubah dengan tidak Logout dari Outlook.com. Agar tetap masuk saat menutup browser, pilih kotak **tetap saya masuk** di halaman login Outlook.com.
