# <a name="my-status-always-shows-as-available"></a>Status saya selalu menunjukkan sebagai tersedia

Setiap kali Anda masuk ke Outlook.com, status account Anda akan reset ke tersedia. Anda dapat mencegah status Anda mengubah dengan tidak menandatangani dari Outlook.com. Untuk tetap masuk ketika menutup peramban, centang kotak **membuatku masuk** pada halaman masuk Outlook.com.
