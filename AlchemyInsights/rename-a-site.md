# <a name="rename-a-sharepoint-site"></a>Ganti nama situs SharePoint

Langkah untuk mengubah/mengganti nama URL situs didokumentasikan dalam [mengubah alamat situs](https://docs.microsoft.com/sharepoint/change-site-address); Misalnya, untuk mengubah dari https://contoso.sharepoint.com/sites/sample1 ke https://contoso.sharepoint.com/sites/sample2 .

Untuk kesalahan yang diketahui saat mengganti nama, lihat [kesalahan ketika Anda mengganti nama alamat situs SharePoint](https://support.office.com/article/errors-when-you-rename-a-sharepoint-site-address-165b7c11-1325-4813-b160-ecbe87bc1a86).

Untuk efek yang dikenal setelah mengubah nama, lihat [efek mengubah alamat situs](https://docs.microsoft.com/sharepoint/change-site-address#effects-of-changing-a-site-address).

**Catatan:** Anda tidak dapat mengubah nama domain (dalam contoh di atas, Bagian "contoso" dari URL) di SharePoint online URL. 

Untuk informasi selengkapnya, lihat:

- [Mengubah nama domain SharePoint](https://go.microsoft.com/fwlink/?Linkid=2018696)
- [Galat saat Anda mengganti nama alamat situs SharePoint](https://support.office.com/article/errors-when-you-rename-a-sharepoint-site-address-165b7c11-1325-4813-b160-ecbe87bc1a86)
