# <a name="rename-a-sharepoint-site"></a>Ganti nama situs SharePoint

Ikuti langkah dalam [artikel ini](https://docs.microsoft.com/sharepoint/change-site-address) untuk mengubah nama situs--misalnya, dari https://contoso.sharepoint.com/sites/sample1 ke. https://contoso.sharepoint.com/sites/sample2

**Catatan:** Anda tidak dapat mengubah nama domain (dalam contoh di atas, contoso bagian dari URL) di SharePoint online URL. 

Untuk informasi selengkapnya, lihat:

- [Mengubah nama domain SharePoint](https://go.microsoft.com/fwlink/?Linkid=2018696)
- [Galat saat Anda mengganti nama alamat situs SharePoint](https://support.office.com/article/errors-when-you-rename-a-sharepoint-site-address-165b7c11-1325-4813-b160-ecbe87bc1a86)
