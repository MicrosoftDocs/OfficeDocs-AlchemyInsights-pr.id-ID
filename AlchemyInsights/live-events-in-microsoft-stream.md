# <a name="live-events-in-microsoft-stream"></a>Acara langsung di Microsoft Stream

Anda dapat membuat acara langsung menggunakan Microsoft Stream di seluruh organisasi. Anda bisa menjadwalkan, menghasilkan, dan menyampaikan acara langsung untuk berbagai skenario seperti acara di seluruh perusahaan, pembaruan kepemimpinan, dan lainnya.

- [Membuat acara langsung di Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Memulai acara langsung Microsoft teams](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Mengonfigurasi enkoder untuk streaming langsung di Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
