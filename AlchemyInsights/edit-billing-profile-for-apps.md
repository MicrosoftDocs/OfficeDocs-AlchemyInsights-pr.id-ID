# <a name="edit-billing-profile-for-apps"></a>Edit profil penagihan untuk aplikasi

## <a name="to-change-the-billing-profile-on-apps"></a>Untuk mengubah profil penagihan Apps

1. Pergi ke **penagihan** > [produk & Jasa](https://go.microsoft.com/fwlink/p/?linkid=842054) halaman, kemudian pilih tab **aplikasi** .

2. Cari, lalu pilih aplikasi yang Anda inginkan untuk mengelola.  

3. Di samping **penagihan profil**, pilih **Edit** dan pilih profil penagihan Anda ingin mengasosiasikan dengan app.

## <a name="edit-billing-profiles"></a>Mengedit profil penagihan

1. Pergi ke **penagihan** > [tagihan pembayaran &](https://go.microsoft.com/fwlink/p/?linkid=848039) halaman, kemudian pilih **penagihan profil**.

2. Memilih profil penagihan dari daftar untuk mengedit.
