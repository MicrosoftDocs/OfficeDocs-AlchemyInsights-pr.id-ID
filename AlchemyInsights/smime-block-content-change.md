# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook kini akan memblokir konten eksternal di S/MIME

Kami memperkenalkan langkah-langkah keamanan yang ditambahkan di Outlook untuk Windows dengan memblokir konten eksternal dalam pesan terenkripsi dan ditandatangani S/MIME secara default dan mencegah menimpa pada setiap pesan.

Untuk detail selengkapnya, lihat [Outlook untuk Windows memblokir konten eksternal dalam pesan S/MIME](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
