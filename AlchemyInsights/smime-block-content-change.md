# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook sekarang akan memblokir default konten eksternal di S/MIME

Kami memperkenalkan langkah keamanan tambahan di Outlook untuk Windows dengan memblokir konten eksternal dalam pesan terenkripsi dan ditandatangani S/MIME secara default dan mencegah penggantian secara per pesan.

Untuk rincian selengkapnya, lihat [Outlook untuk Windows memblokir konten eksternal dalam pesan S/MIME](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
