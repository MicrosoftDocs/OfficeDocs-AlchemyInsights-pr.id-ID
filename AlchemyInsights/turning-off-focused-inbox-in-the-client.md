# <a name="turn-off-focused-inbox-in-email-clients"></a>Matikan focused Inbox di klien email

Untuk menonaktifkan kotak masuk fokus di Outlook 2016, Outlook di web, Outlook.com/Hotmail.com, Outlook 2016 untuk Mac, atau mail untuk Windows 10, lihat menonaktifkan [kotak masuk fokus](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Untuk menonaktifkan kotak masuk fokus untuk semua orang di organisasi, lihat [mengkonfigurasi fokus kotak masuk untuk semua orang di organisasi Anda](https://docs.microsoft.com/office365/admin/setup/configure-focused-inbox).
