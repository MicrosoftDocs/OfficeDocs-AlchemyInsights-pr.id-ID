# <a name="pst-data-files-in-outlookcom"></a>File data PST di Outlook.com

Tidak dimungkinkan untuk mengimpor Outlook. File data PST ke akun Outlook.com Anda menggunakan browser.

Untuk mengimpor. File data PST, Anda harus menggunakan aplikasi desktop Outlook. Untuk mempelajari selengkapnya, baca [mengimpor email, kontak, dan kalender dari Outlook. pst.](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)
