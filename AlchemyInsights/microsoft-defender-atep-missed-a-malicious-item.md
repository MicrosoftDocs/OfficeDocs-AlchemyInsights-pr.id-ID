# <a name="microsoft-defender-atp-missed-a-malicious-item"></a>Microsoft Defender ATP melewatkan item berbahaya

- Kirimkan berkas berbahaya ke Microsoft untuk analisis pada [Microsoft Defender respons portal](https://www.microsoft.com/wdsi/filesubmission/). 
- Jika Anda masuk, temukan Riwayat pengiriman berkas di [portal respons Microsoft Defender](https://www.microsoft.com/wdsi/submissionhistory).
- Untuk informasi lebih lanjut, lihat [mengirimkan file untuk dianalisis](https://docs.microsoft.com/windows/security/threat-protection/intelligence/submission-guide).
