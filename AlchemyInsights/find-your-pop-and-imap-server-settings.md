# <a name="find-your-pop-or-imap-server-settings"></a>Menemukan pengaturan server POP atau IMAP Anda

Jika tidak mengetahui pengaturan yang diperlukan untuk menambahkan akun POP atau IMAP ke Outlook, Anda dapat mencarinya di [pengaturan email pop dan IMAP untuk Outlook](https://support.office.com/article/8361e398-8af4-4e97-b147-6c6c4ac95353.aspx).
  
> [!NOTE]
> Apakah Anda tahu bahwa Anda bisa [mengaktifkan dan mengonfigurasikan POP3 di server Exchange 2016](https://technet.microsoft.com/library/bb124934%28v=exchg.160%29.aspx)?
