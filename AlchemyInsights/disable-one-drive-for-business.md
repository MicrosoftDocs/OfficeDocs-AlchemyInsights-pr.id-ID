# <a name="disable-onedrive-for-business"></a>Menonaktifkan OneDrive for Business

Ada beberapa cara untuk menonaktifkan akses ke OneDrive dan layanannya tergantung pada kebutuhan bisnis Anda:

- [Menonaktifkan, menonaktifkan, atau Menghapus instalan OneDrive](https://support.office.com/article/turn-off-disable-or-uninstall-onedrive-f32a17ce-3336-40fe-9c38-6efb09f944b0)
- [Izinkan sinkronisasi akun OneDrive untuk hanya organisasi tertentu](https://docs.microsoft.com/onedrive/use-group-policy#allow-syncing-onedrive-accounts-for-only-specific-organizations)
- [Menonaktifkan pembuatan situs OneDrive untuk beberapa pengguna](https://docs.microsoft.com/sharepoint/manage-user-profiles#disable-onedrive-creation-for-some-users)
- [Mengontrol akses ke data SharePoint online dan OneDrive berdasarkan lokasi jaringan](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location)
- [Mencegah pengguna untuk menyinkronkan akun OneDrive pribadi](https://docs.microsoft.com/onedrive/use-group-policy#DisablePersonalSync)
- [Mengelola koneksi dari perangkat Windows ke OneDrive dan layanan lainnya](https://docs.microsoft.com/windows/privacy/manage-connections-from-windows-operating-system-components-to-microsoft-services#bkmk-onedrive)
