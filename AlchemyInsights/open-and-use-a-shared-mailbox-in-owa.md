# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>Membuka dan menggunakan kotak surat bersama di Outlook di web

Kotak surat bersama yang dapat Anda akses tidak otomatis ditampilkan di Outlook di web (sebelumnya dikenal sebagai Outlook Web App).

Anda dapat menampilkan kotak surat bersama dan kotak surat utama secara bersamaan atau membuka kotak surat bersama di jendela browser terpisah. Untuk menerima pemberitahuan pesan baru pada kotak surat bersama, Anda harus membuka kotak surat bersama di jendela browser terpisah. Untuk informasi selengkapnya, lihat [Membuka dan menggunakan kotak surat bersama di Outlook Web App](https://support.office.com/id-ID/article/Add-a-shared-mailbox-to-Outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
