# <a name="microsoft-365-for-nonprofits---approved"></a>Microsoft 365 untuk nirlaba-disetujui

Status Anda sebagai organisasi nirlaba berkualitas Microsoft telah disetujui. Anda dapat menyelesaikan pembelian dari Pusat admin Microsoft 365.

1. Di pusat admin, masuk ke **Billing** \> halaman [Layanan pembelian](https://go.microsoft.com/fwlink/p/?linkid=868433) tagihan.

2. Pilih dari daftar di bawah **paket nirlaba**.

3. Arahkan mouse ke langganan yang ingin Anda beli, lalu pilih **Beli sekarang**.

Untuk informasi selengkapnya, lihat [sumber daya Microsoft 365 untuk nirlaba](https://www.microsoft.com/nonprofits/microsoft-365) atau [Hubungi penjualan](https://www.microsoft.com/nonprofits/contact-us).
