# <a name="manage-who-can-create-microsoft-365-groups"></a>Mengelola siapa yang dapat membuat Microsoft 365 grup

Secara default, siapa pun di organisasi Anda dapat membuat grup O365.
  
Jika Anda ingin mengubahnya, Anda dapat membuat grup aman dengan PowerShell yang membatasi pembuatan grup ke subset pengguna.
  
Ikuti petunjuk langkah demi langkah di sini: [mengelola siapa yang dapat membuat Microsoft 365 grup](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
