# <a name="create-users-in-hybrid-deployments"></a>Membuat pengguna dalam penggunaan hibrid

Ketika organisasi Anda hibrid antara lingkungan lokal dan 365 Microsoft, Anda perlu membuat semua objek pengguna di direktori aktif di tempat Anda. Anda membuat objek pengguna di unit organisasi yang sesuai (OU) sehingga mereka telah disinkronkan ke organisasi Anda.
