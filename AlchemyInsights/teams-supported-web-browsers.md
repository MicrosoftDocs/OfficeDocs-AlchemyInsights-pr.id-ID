# <a name="teams-supported-web-browsers"></a>Web browser yang didukung oleh tim

Tim sepenuhnya mendukung browser internet yang tercantum dalam artikel, [batas dan spesifikasi untuk Microsoft teams](https://docs.microsoft.com/microsoftteams/limits-specifications-teams#browsers), dengan pengecualian yang dicatat untuk panggilan dan pertemuan.
