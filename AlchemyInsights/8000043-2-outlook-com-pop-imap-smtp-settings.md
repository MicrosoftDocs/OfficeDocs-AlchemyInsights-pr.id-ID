# <a name="pop-imap-and-smtp-settings-for-outlookcom"></a>Pengaturan POP, IMAP, dan SMTP untuk Outlook.com

Jika Anda ingin menambahkan Outlook.com account lain aplikasi e-mail yang mendukung POP atau IMAP, berikut adalah pengaturan manual server yang Anda perlukan:

- **Nama server IMAP**: imap-mail.outlook.com
- **IMAP port**: 993
- **Nama server POP**: pop-mail.outlook.com
- **POP port**: 995
- **POP metode enkripsi**: TLS
- **Nama server SMTP**: smtp-mail.outlook.com
- **SMTP port**: 587
- **Metode enkripsi SMTP**: STARTTLS

Baca lebih lanjut tentang [pengaturan POP, IMAP, dan SMTP untuk Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001402&clcid=0x409).
