# <a name="how-to-right-click-on-a-mac"></a>Cara klik kanan pada Mac

Di komputer Mac, klik kanan dikenal sebagai klik sekunder atau klik kontrol. Jika mouse, trackpad, atau perangkat input lainnya tidak menyertakan tombol klik kanan atau cara lain untuk melakukan klik kanan, tahan tombol **kontrol** pada keyboard saat Anda mengeklik.
  
Untuk informasi lebih lanjut, pergi ke [dukungan Apple](https://go.microsoft.com/fwlink/?linkid=2022220&amp;clcid=0x409).
