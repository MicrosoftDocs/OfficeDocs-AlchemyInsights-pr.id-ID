# <a name="distribution-group-not-showing-in-groups-list"></a>Grup distribusi tidak ditampilkan dalam daftar Grup

Dapat memakan waktu hingga 60 menit sebelum daftar distribusi baru muncul di pusat pusat admin Exchange.
  
Jika Anda ingin menambahkan anggota atau melakukan pengeditan lain sekarang, silakan gunakan fitur grup di [Pusat admin Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx).
