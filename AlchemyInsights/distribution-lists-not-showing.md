# <a name="distribution-group-not-showing-in-groups-list"></a>Grup distribusi yang tidak menunjukkan dalam daftar Grup

Ini dapat mengambil hingga 60 menit sebelum daftar distribusi baru muncul di pusat pusat admin asing.
  
Jika Anda ingin menambahkan anggota atau membuat suntingan lain sekarang, silakan gunakan fitur grup di [Asing Admin Center](https://support.office.com/article/https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx).
