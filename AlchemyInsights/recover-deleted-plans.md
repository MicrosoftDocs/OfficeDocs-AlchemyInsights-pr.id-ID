# <a name="recover-deleted-plans"></a>Memulihkan paket yang dihapus

- Hanya pemilik paket yang dapat menghapus paket. Untuk [menghapus paket atau tugas](https://support.microsoft.com/id-ID/office/delete-a-task-or-plan-39e10e78-13f0-446d-94cd-9e562648497a.), ikuti instruksi untuk menghapus tugas atau paket.  **Catatan**: Jika paket dihapus secara permanen, paket tidak dapat dipulihkan. Demikian juga, tidak ada cara untuk memulihkan tugas yang dihapus. Jika secara tidak sengaja menghapus sebuah tugas, Anda harus membuat ulang dari awal.

- Jika paket Anda telah dihapus akibat menghapus grup, Anda mungkin dapat memulihkan grup beserta kontennya. Ikuti instruksi untuk memulihkan [grup Office 365 yang dihapus](https://docs.microsoft.com/microsoft-365/admin/create-groups/restore-deleted-group?view=o365-worldwide) untuk memulihkan grup dan konten terkait.

- Jika Anda menghapus tab Planner dari saluran Teams tanpa menghapus paket secara permanen, Anda dapat menambahkan tab Planner dan menambahkan paket kembali dengan masuk ke **Menambahkan Tab > Planner > Gunakan paket yang sudah ada dari Tim ini**, lalu pilih paket dari menu menurun.
