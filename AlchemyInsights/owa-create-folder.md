# <a name="create-a-folder"></a>Membuat folder

1. Di bagian bawah panel folder di Outlook di web, pilih **folder baru**. Anda mungkin perlu menggulir ke bawah di panel folder untuk melihat tombol **folder baru** .
1. Dalam kotak teks, masukkan nama untuk folder tersebut lalu tekan Enter.

Untuk mempelajari selengkapnya, lihat [bekerja dengan folder pesan di Outlook di web](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
