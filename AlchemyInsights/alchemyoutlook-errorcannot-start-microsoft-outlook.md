# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Galat Outlook: "tidak dapat memulai Microsoft Outlook"

Apakah Anda mengalami masalah memulai Outlook 2016, Outlook 2013, atau Outlook 2010, atau menerima "tidak dapat memulai Microsoft Outlook. Tidak dapat membuka jendela Outlook "pesan galat?
  
Pertama, gunakan langkah pemecahan masalah dalam [artikel ini](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) untuk membantu menyelesaikan masalah startup ini. 
  
Anda juga dapat menjalankan [alat diagnostik lanjut](https://aka.ms/SaRA-OutlookAdvDiagnostics) pada mesin yang terpengaruh.
