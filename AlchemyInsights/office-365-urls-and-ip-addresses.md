# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365 URL dan kisaran alamat IP

Microsoft 365 memerlukan konektivitas ke Internet melalui alamat IP tertentu yang disebut titik *akhir*.
Untuk daftar akhir yang harus dicapai, lihat [Microsoft 365 URL dan kisaran alamat IP](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Layanan web Microsoft 365 IP address dan URL membantu Anda mengidentifikasi dan membedakan lalu lintas jaringan 365 Microsoft dengan lebih baik, sehingga memudahkan Anda untuk mengevaluasi, mengonfigurasi, dan tetap mengikuti perubahan. Lihat [Office 365 alamat IP dan layanan web URL](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
