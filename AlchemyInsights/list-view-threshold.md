# <a name="list-view-threshold"></a>Ambang batas tampilan daftar

Ambang tampilan daftar berada di tempat untuk mempertahankan kinerja layanan. Ambang tidak dapat diubah di SharePoint online. Untuk menghindari mencapai ambang batas ini, lihat [mengelola daftar besar dan Perpustakaan di SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59).

Anda mungkin juga menerima kesalahan ini saat mencoba membagikan atau menghentikan pewarisan folder. Melihat [galat berbagi/Break warisan](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance).
