# <a name="list-view-threshold"></a>Ambang batas tampilan daftar

Daftar dan pustaka SharePoint dapat memuat hingga 30 juta item. Namun, jika berisi lebih dari 5.000 item, Anda mungkin melihat kesalahan ambang batas tampilan daftar saat mencoba mengerjakan item tersebut. Ambang batas ini ada untuk mempertahankan kinerja layanan. Ambang batas ini tidak dapat diubah. Anda mungkin juga menerima kesalahan ini saat mencoba membagikan atau menghentikan pewarisan folder.

Agar tidak melampaui ambang batas ini, buka:

- [Mengelola daftar dan pustaka berukuran besar di SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59)

- [Kesalahan Membagikan/Menghentikan Pewarisan](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance)
