# <a name="error-onedrive-cannot-connect-to-windows"></a>Kesalahan: OneDrive tidak dapat tersambung ke Windows

Untuk mengatasi pesan kesalahan, Cari file cldflt.sys untuk melihat apakah file berjalan, dan jika tidak, Aktifkan file tersebut. 
  
Untuk informasi tambahan tentang cara mengaktifkan file cldflt.sys, lihat [kesalahan OneDrive tidak dapat tersambung ke Windows "ketika mengakses file di OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
