# <a name="does-visio-work-on-a-mac"></a>Apakah Visio berfungsi di Mac?

Aplikasi desktop Visio tidak tersedia untuk Mac. Namun, selama pengguna yang membutuhkan aplikasi memiliki [lisensi Visio online yang ditetapkan](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users), mereka dapat [melihat, membuat, dan mengedit diagram di Visio online](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) di browser web.
