# <a name="update-dns-records-to-keep-your-website-with-your-current-hosting-provider"></a>Pembaruan data DNS untuk menjaga website Anda dengan penyedia hosting Anda saat ini

Sebagai bagian dari proses pindahan kantor 365 umum Website Anda ke penyedia hosting lain, Anda mungkin perlu membuat perubahan pada catatan domain Anda.
  
1. Di pusat admin, pergi ke **pengaturan** \> **domain** dan memilih nama domain yang ingin Anda ubah. 
    
2. Masukkan informasi yang sesuai untuk situs web Anda dengan penyedia hosting Anda baru.
    
Untuk selengkapnya, lihat [pembaruan data DNS untuk menjaga website Anda dengan penyedia hosting Anda saat ini](https://support.office.com/article/update-dns-records-to-keep-your-website-with-your-current-hosting-provider-2c4cf347-b897-45c1-a71f-210bdc8f1061). 
  
Untuk mempelajari lebih lanjut tentang penghentian situs umum, lihat [SharePoint Online situs-web publik akan dihentikan](https://support.office.com/article/sharepoint-online-public-websites-to-be-discontinued-e86bfd2f-5c7d-446f-a430-7cfcc0130916?ui=en-US&amp;rs=en-US&amp;ad=US).
