# <a name="manage-groups-in-yammer"></a>Mengelola grup di Yammer

- **Izin pembuatan grup.** Heboh tidak mengontrol siapa yang dapat membuat grup. Namun, dimungkinkan untuk mengelola pembuatan grup untuk terhubung Microsoft 365 grup. Lihat [mengelola siapa yang dapat membuat Microsoft 365 grup](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups) untuk informasi lebih lanjut.

- **Keanggotaan grup dinamis.** Silakan merujuk ke [membuat grup dinamis di heboh](https://docs.microsoft.com/yammer/manage-yammer-groups/create-a-dynamic-group) untuk informasi tentang membuat keanggotaan dinamis untuk heboh terhubung Microsoft 365 grup.

- **Memulihkan grup yang dihapus.** Memulihkan heboh kelompok dimungkinkan melalui [pemulihan dihapus Microsoft 365 grup](https://docs.microsoft.com/microsoft-365/admin/create-groups/restore-deleted-group) non terhubung kelompok tidak dapat dipulihkan setelah penghapusan.

Untuk informasi lebih lanjut tentang mengelola grup, lihat [mengelola grup di heboh](https://support.office.com/article/Manage-a-group-in-Yammer-6e05c6d6-5548-4c88-89cd-e6757a514ef2). Untuk informasi tentang terhubung Microsoft 365 grup, lihat [heboh dan microsoft 365 grup](https://docs.microsoft.com/yammer/manage-yammer-groups/yammer-and-office-365-groups).
