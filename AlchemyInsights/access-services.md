# <a name="access-services"></a>Layanan akses

Fitur layanan akses di SharePoint online sedang dihentikan. Baru berbasis akses Web Apps dan web database tidak dapat lagi dibuat, dan ada Web Apps dan web database akan ditutup oleh April, 2018. Akses pangkalan data desktop (. accdb) tidak dipengaruhi oleh perubahan ini. Kami merekomendasikan Microsoft PowerApps sebagai alternatif untuk mengakses Web Apps dan database web. 
  
[Pelajari tentang Microsoft PowerApps](https://powerapps.microsoft.com/)
