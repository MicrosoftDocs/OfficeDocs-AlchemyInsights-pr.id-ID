# <a name="access-services"></a>Layanan Access

Fitur Access Services di SharePoint online akan dihentikan. Aplikasi web dan database web berbasis akses baru tidak dapat dibuat lagi, dan database web dan aplikasi web yang ada akan ditutup pada April, 2018. Access database desktop (. accdb) tidak terpengaruh oleh perubahan ini. Kami merekomendasikan Microsoft PowerApps sebagai alternatif untuk mengakses aplikasi web dan database web. 
  
[Pelajari tentang Microsoft PowerApps](https://powerapps.microsoft.com/)
