# <a name="access-services"></a>Layanan akses

Fitur layanan akses dalam SharePoint Online sedang dihentikan. Baru aplikasi web berbasis akses dan database web tidak lagi dapat dibuat, dan ada aplikasi web dan database web akan menutup oleh April, 2018. Akses Desktop database (.accdb) tidak terpengaruh oleh perubahan ini. Kami merekomendasikan Microsoft PowerApps sebagai alternatif untuk akses web apps dan database web. 
  
[Pelajari tentang Microsoft PowerApps](https://powerapps.microsoft.com/)
