# <a name="verify-your-domain"></a>Memverifikasi domain Anda

 **Catatan mungkin belum diperbarui di seluruh internet.**
  
Biasanya hanya memerlukan beberapa menit agar kami dapat melihat catatan baru, tapi terkadang dapat memakan waktu selama beberapa jam. 
  
- Jika sudah lama menunggu, periksa ulang Apakah Anda telah menyalin dan menempelkan nilai persis ke dalam catatan verifikasi TXT di host DNS Anda. Salah satu masalah umum tidak termasuk "MS =" bagian dari catatan. Kami juga membutuhkannya!

- Di beberapa host DNS, Anda harus mengambil langkah tambahan untuk menyimpan file zona (tempat catatan DNS disimpan) sehingga akan diperbarui di seluruh internet. Pastikan Anda telah menyimpan perubahan Anda sehingga Microsoft bisa melihat dan memverifikasi catatan.
