# <a name="connect-to-the-mscommerce-module"></a>Menyambung ke modul MSCommerce

Anda harus tersambung ke modul MSCommerce sebelum Anda dapat melihat atau menetapkan kebijakan AllowSelfServicePurchase.  

Untuk menyambung ke modul MSCommerce, pada prompt PowerShell (PS C:\), masukkan perintah berikut:

    Connect-MSCommerce

Ini akan membuka dialog masuk. Masukkan nama pengguna dan sandi untuk masuk.

**Catatan:**&nbsp;&nbsp;akun yang digunakan untuk masuk harus merupakan perusahaan atau administrator penagihan.
