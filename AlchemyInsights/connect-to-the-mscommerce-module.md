# <a name="connect-to-the-mscommerce-module"></a>Menyambungkan ke modul MSCommerce

Anda harus tersambung ke modul MSCommerce sebelum Anda bisa menampilkan atau mengatur kebijakan AllowSelfServicePurchase.  

Untuk menyambungkan ke modul MSCommerce, pada prompt PowerShell (PS C: \) , masukkan perintah berikut:

`Connect-MSCommerce`

Ini akan membuka dialog masuk. Masukkan nama pengguna dan kata sandi Anda untuk masuk.

**Catatan:** &nbsp; &nbsp; Akun yang digunakan untuk masuk harus merupakan perusahaan atau administrator tagihan.
