# <a name="whats-a-hub-site"></a>Apa itu situs hub?

Situs hub membantu Anda menata situs dan memberi mereka navigasi dan pencitraan umum. Admin global atau SharePoint akan dapat membuat situs hub dan menentukan pengguna mana yang dapat mengaitkan situs mereka dengan situs hub. Untuk informasi selengkapnya, lihat [situs SharePoint hub](https://go.microsoft.com/fwlink/?linkid=869388).
