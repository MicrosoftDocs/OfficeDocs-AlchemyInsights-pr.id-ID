# <a name="one-of-your-on-premises-federation-service-certificates-is-expiring"></a>Salah satu sertifikat Layanan Federasi lokal Anda kedaluwarsa

Untuk mengatasi masalah ini, ikuti langkah berikut:
  
- Menginstal Microsoft Azure aktif direktori modul untuk Windows PowerShell di komputer (jika tidak menginstal modul). Untuk melakukan ini, pergi ke [Azure PowerShell direktori aktif untuk grafik](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)
    
- Ikuti langkah-langkah dalam "skenario 1: AD FS token-penandatanganan sertifikat kadaluarsa" bagian ["Ada masalah mengakses situs" kesalahan dari AD FS ketika pengguna Federasi masuk ke kantor 365, Azure, atau Intune](https://support.microsoft.com/help/2713898/there-was-a-problem-accessing-the-site-error-from-ad-fs-when-a-federat).
    
- Ikuti langkah-langkah dalam t[cara memperbarui atau memperbaiki pengaturan domain federasi di Office 365, Azure, atau Intune](https://support.microsoft.com/help/2647048/how-to-update-or-repair-the-settings-of-a-federated-domain-in-office-3).
    
Untuk informasi selengkapnya tentang memperbarui sertifikat Federasi, lihat [pembaruan sertifikat untuk O365 dan Azure iklan](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-o365-certs).
