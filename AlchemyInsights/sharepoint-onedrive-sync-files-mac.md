# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Instal, setup, dan sinkronkan file OneDrive atau SharePoint di Mac 

Menginstal, setup dan Sync OneDrive atau SharePoint file pada Mac dengan instruksi dari [file Sync dengan OneDrive sinkronisasi klien di Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Jika aplikasi OneDrive diinstal dan penataan di Mac, tetapi pengguna mengalami masalah sinkronisasi, lihat [memperbaiki masalah sinkronisasi dengan OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756).

> [!NOTE]
> Fitur memindahkan folder yang diketahui di sinkronisasi OneDrive untuk Windows tidak tersedia untuk Mac.
