# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Laporan file dan folder berbagi di situs SharePoint

Anda dapat membuat file CSV yang merinci setiap file, pengguna, izin, dan link unik di situs SharePoint tertentu atau Perpustakaan OneDrive. Hal ini dapat membantu Anda memahami cara berbagi digunakan dan apakah ada file atau folder yang dibagikan dengan tamu.

Untuk informasi selengkapnya, [laporan berbagi berkas dan folder di situs SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Fitur ini saat ini digunakan untuk penyewa rilis pertama dan akan secara bertahap bergulir ke semua penyewa produksi.
