# <a name="learn-about-microsoft-365-groups"></a>Pelajari tentang Microsoft 365 grup

Microsoft 365 Groups adalah layanan yang berfungsi dengan alat Office 365 yang sudah Anda gunakan, sehingga Anda dapat berkolaborasi dengan rekan tim saat menulis dokumen, membuat spreadsheet, mengerjakan rencana proyek, menjadwalkan Rapat, atau mengirim email. Untuk informasi selengkapnya, lihat [Pelajari tentang Microsoft 365 grup](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
