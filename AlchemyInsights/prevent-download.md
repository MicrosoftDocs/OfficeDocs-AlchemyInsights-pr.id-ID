# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Mencegah file diunduh dari SharePoint atau OneDrive

Terdapat berbagai metode yang mungkin digunakan untuk mencegah atau memblokir unduhan yang terjadi. Silakan kunjungi link di bawah ini untuk masing-masing metode:

- [Memblokir unduhan file dengan akses bersyarat Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Mengontrol akses dari perangkat yang tidak dikelola](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Mengonfigurasi kebijakan IRM pada pustaka dokumen SharePoint untuk membatasi pengunduhan konten](https://docs.microsoft.com/microsoft-365/compliance/set-up-irm-in-sp-admin-center)
