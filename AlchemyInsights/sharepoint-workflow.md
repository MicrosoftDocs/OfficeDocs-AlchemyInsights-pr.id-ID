# <a name="troubleshoot-workflows-in-sharepoint"></a>Memecahkan masalah alur kerja di SharePoint

**Memecahkan masalah dan alur kerja debug di SharePoint**

Lihat [memecahkan masalah galat validasi alur kerja SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio) dan [debugging alur kerja SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).

Jika Anda baru memulai dengan alur kerja, pertimbangkan untuk menggunakan [daya otomatis](https://docs.microsoft.com/power-automate/modern-approvals) daripada solusi alur kerja SharePoint klasik.

Untuk informasi selengkapnya, lihat [alur kerja SharePoint pensiun](https://docs.microsoft.com/alchemyinsights/sharepoint-workflows-retiring).
