# <a name="create-a-shared-mailbox"></a>Membuat kotak surat bersama

Kotak surat bersama memungkinkan lebih dari satu orang mengirim dan menerima email untuk alias, seperti sales@contoso.com, tanpa menetapkan lisensi lain. 

Lihat artikel [membuat kotak surat bersama](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) untuk Tips dan Info lebih lanjut tentang menggunakan kotak surat bersama.
