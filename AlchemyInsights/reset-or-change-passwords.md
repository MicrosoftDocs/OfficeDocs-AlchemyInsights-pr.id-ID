# <a name="reset-or-change-passwords"></a>Menyetel ulang atau mengubah sandi

Pengguna dapat mengubah sandi mereka sendiri dari **akun** > saya[keamanan & privasi](https://portal.office.com/account/#security).
  
Jika pengguna lupa sandinya, admin dapat menyetel ulang sandi dengan membuka > pengguna[aktif](https://portal.office.com/adminportal/home#/users)pengguna, **Users**pilih pengguna, dan klik **setel ulang sandi**.
  
[Klik di sini untuk mempelajari lebih lanjut](https://docs.microsoft.com/office365/admin/add-users/reset-passwords), atau membiarkan pengguna me-reset password mereka sendiri dilupakan.
