# <a name="files-restore-feature"></a>Fitur Pemulihan File

Jika banyak file OneDrive atau SharePoint yang terhapus, ditimpa, rusak, atau terinfeksi oleh malware, Anda dapat memulihkan keseluruhan pustaka OneDrive atau SharePoint ke waktu sebelumnya menggunakan fitur Pemulihan File. Untuk informasi selengkapnya, lihat:

- [Memulihkan pustaka OneDrive](https://support.office.com/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a150)

- [Memulihkan pustaka dokumen](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)
