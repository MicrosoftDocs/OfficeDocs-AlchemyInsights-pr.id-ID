# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Ubah pengaturan bahasa tampilan dalam SharePoint atau OneDrive 

<p>Anda tidak dapat mengubah<strong> pengaturan bahasa default</strong> untuk SharePoint atau OneDrive karena pengaturan tersebut telah ditetapkan saat membuat Anda subcription Office 365. &nbsp;Anda dapat, namun, mengubah setelan bahasa <strong>tampilan</strong> .&nbsp;</p> <ol> <li><span style="display: inline !important; float: none; cursor: text; font-family: Verdana,Arial,Helvetica,sans-serif; font-size: 12px; font-style: normal; font-variant: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-decoration: none; text-indent: 0px; text-transform: none; -webkit-text-stroke-width: 0px; white-space: normal; word-spacing: 0px;">Untuk mengubah pengaturan pribadi Anda</span>, lihat <a title="mengubah setelan bahasa dan wilayah pribadi" href="https://support.office.com/en-us/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7">Mengubah setelan bahasa dan wilayah pribadi</a>.&nbsp;<br /><br /></li> <li>Untuk mengubah antarmuka pengguna situs, lihat <a title="memilih bahasa yang Anda inginkan untuk membuat tersedia untuk antarmuka pengguna situs" href="https://support.office.com/en-us/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8">Pilih bahasa yang Anda inginkan untuk membuat tersedia untuk antarmuka pengguna situs</a>. Untuk selengkapnya tentang memecahkan masalah pengaturan bahasa, lihat <a title="Anda tidak dapat mengubah bahasa default untuk SharePoint Online situs koleksi" href="https://support.office.com/en-us/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964">Anda tidak dapat mengubah bahasa default untuk SharePoint Online situs koleksi</a>.</li> </ol> <p>&nbsp;</p>
