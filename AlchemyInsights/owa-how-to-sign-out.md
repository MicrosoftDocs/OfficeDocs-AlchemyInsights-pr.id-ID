# <a name="how-to-sign-out-of-outlook-on-the-web"></a>Cara sign out dari Outlook di web

1. Pilih nama Anda atau gambar di sudut kanan atas.
    
    Jika Anda tidak melihat gambar Anda di bagian atas layar, periksa untuk melihat jika iklan memblokir diaktifkan. Kadang-kadang ini mencegah gambar menampilkan.
    
2. Pilih **keluar**. 
    
    Untuk berhenti secara otomatis masuk ke pekerjaan Anda atau sekolah account, Hapus centang pada kotak untuk **menjaga saya masuk** pada halaman di mana Anda memasukkan password Anda. 
    
Untuk selengkapnya, lihat [cara masuk atau keluar dari Outlook di web](https://support.office.com/article/763fab4d-0138-4814-b450-37fc286bcb79).
