# <a name="use-your-browser-to-check-spelling"></a>Gunakan browser Anda untuk Periksa ejaan

Ada tidak ada spell check pilihan di Outlook di web. Untuk memeriksa ejaan, Anda akan perlu menggunakan web browser. Ejaan tersedia di Microsoft Edge, Internet Explorer 10 dan versi, dan versi terbaru Firefox, Chrome dan Safari. Periksa pilihan browser web Anda untuk mempelajari lebih lanjut tentang cara memeriksa ejaan.
