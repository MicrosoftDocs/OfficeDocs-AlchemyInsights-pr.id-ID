# <a name="upload-a-microsoft-teams-meeting-recording-to-stream-a1"></a>Mengunggah perekaman Rapat Microsoft teams ke Stream (A1)

Untuk sebagian besar pengguna Microsoft 365, rekaman Rapat teams diunggah secara otomatis ke Microsoft Stream setelah rapat. Namun, mulai  **20 agustus 2020**, ketika pengguna dengan lisensi A1 (termasuk A1 Plus) memulai rekaman Rapat teams, perilaku ini berubah.  

Untuk informasi selengkapnya, lihat [mengunggah rekaman Rapat Microsoft teams untuk streaming](https://docs.microsoft.com/stream/portal-upload-teams-meeting-recording).
