# <a name="screen-sharing-with-skype-consumer-users"></a>Berbagi layar dengan pengguna Skype

Untuk instruksi umum tentang cara berbagi program Anda, untuk mempelajari tentang larangan berbagi, dan untuk pemecahan masalah sumber daya, Tinjau [berbagi layar Anda di Skype for Business](https://support.microsoft.com/office/share-and-present-content-from-skype-meetings-app-skype-for-business-web-app-234b0c06-a88d-4707-904c-4fd6c571fc01).  

Saat ini, pengguna Skype for Business tidak dapat berbagi layar dengan pengguna Skype. [Klik di sini](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) untuk memberikan umpan balik tentang mengaktifkan berbagi layar dengan pengguna Skype konsumen.
