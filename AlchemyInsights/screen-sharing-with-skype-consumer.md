# <a name="screen-sharing-with-skype-consumer-users"></a>Berbagi layar dengan pengguna konsumen Skype

Untuk petunjuk umum tentang cara berbagi program, untuk mempelajari tentang pembatasan dengan berbagi, dan untuk pemecahan masalah sumber daya, Tinjau [berbagi layar Anda di Skype untuk bisnis](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

Saat ini, Skype untuk pengguna bisnis tidak dapat layar berbagi dengan Skype pengguna konsumen. Silahkan [Klik di sini](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) untuk memberikan umpan balik mengenai mengaktifkan berbagi layar dengan konsumen Skype pengguna.
