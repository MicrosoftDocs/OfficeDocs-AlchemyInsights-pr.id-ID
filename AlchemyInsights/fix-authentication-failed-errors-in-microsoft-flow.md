# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Perbaiki galat "otentikasi gagal" di Microsoft Flow

Jika Anda menerima pesan galat "otentikasi gagal" saat Anda mencoba masuk ke Microsoft Flow, hal ini karena layanan aliran dinonaktifkan di Azure Active Directory. Untuk detail dan langkah untuk memperbaiki kesalahan, lihat [Anda tidak dapat masuk ke Microsoft Flow dan Anda menerima galat "otentikasi gagal"](https://support.microsoft.com/help/4316891).
