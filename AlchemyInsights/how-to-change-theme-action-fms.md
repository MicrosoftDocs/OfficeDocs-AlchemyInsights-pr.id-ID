# <a name="how-to-change-your-theme"></a>Bagaimana untuk mengubah tema

Untuk mengubah tema Anda di Outlook.com atau Outlook di web, pergi ke **pengaturan** dan pilih tema **tema**. Untuk melihat semua tema yang tersedia, pilih **Lihat semua** **tema**.
