# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Me-refresh menggunakan Web connector tidak bekerja dengan benar

Jika Anda memiliki konektor web script yang menggunakan fungsi [Web.Page](https://msdn.microsoft.com/library/mt260924.aspx) , dan Anda telah memperbarui dataset atau laporan Anda setelah tanggal 18 November 2016, Anda perlu menggunakan gateway agar refresh untuk bekerja dengan baik.

Untuk informasi lebih lanjut:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
