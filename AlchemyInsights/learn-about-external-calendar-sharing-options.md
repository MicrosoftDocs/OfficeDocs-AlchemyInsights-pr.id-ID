# <a name="learn-about-external-calendar-sharing-options"></a>Pelajari tentang opsi berbagi kalender eksternal

Jika Anda perlu menjadwalkan pertemuan dengan orang di luar organisasi Anda, Microsoft 365 memungkinkan Anda membuat kalender tersedia untuk "pengguna eksternal", mereka yang perlu melihat waktu bebas/sibuk namun tidak memiliki akun pengguna untuk lingkungan Microsoft 365 Anda. Untuk informasi lebih lanjut, lihat [berbagi kalender dengan pengguna eksternal](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
