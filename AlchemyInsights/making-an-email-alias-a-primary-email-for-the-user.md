# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>Membuat alias email sebagai alamat utama untuk pengguna

Untuk petunjuknya, lihat [menambahkan alias email lain untuk pengguna](https://docs.microsoft.com/office365/admin/email/add-another-email-alias-for-a-user).

**Catatan**: jika organisasi Anda berada dalam mode hibrida, Anda perlu mengubah alamat email utama di organisasi Exchange lokal Anda. Untuk selengkapnya, lihat [menambahkan atau menghapus alamat email untuk kotak pesan](https://technet.microsoft.com/library/bb123794.aspx).
