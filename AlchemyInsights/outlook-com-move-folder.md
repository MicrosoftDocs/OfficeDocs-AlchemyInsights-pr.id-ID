# <a name="move-a-folder-in-outlookcom"></a>Memindahkan folder di Outlook.com

Anda bisa mengurutkan ulang folder di bawah **favorit**, tapi Anda tidak bisa mengurutkan ulang folder di bawah **folder** atau membuat subfolder menjadi folder. Jika Anda secara tidak sengaja memindahkan folder ke folder lain, Buat folder baru dan Salin semua email Anda dari subfolder ke folder baru.
  
Untuk mempelajari selengkapnya, baca [bekerja dengan folder pesan di Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
