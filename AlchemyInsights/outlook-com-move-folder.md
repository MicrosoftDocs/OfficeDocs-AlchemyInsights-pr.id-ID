# <a name="move-a-folder-in-outlookcom"></a>Memindahkan folder di Outlook.com

Anda dapat menyusun folder di bawah **favorit**, tetapi Anda tidak dapat menyusun ulang folder di bawah **folder** atau membuat subfolder folder. Jika Anda tidak sengaja memindahkan folder ke folder lain, Buat folder baru dan Salin semua email dari subfolder ke folder baru.
  
Untuk mempelajari lebih lanjut, baca [bekerja dengan folder pesan di Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
