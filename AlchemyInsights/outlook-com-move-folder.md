# <a name="move-a-folder"></a>Memindahkan folder

Anda dapat menyusun folder di bawah **favorit**, tetapi Anda tidak dapat menyusun ulang folder di bawah **folder** atau buat subfolder folder. Jika Anda secara tidak sengaja memindahkan folder ke folder lain, membuat folder baru dan menyalin semua email dari subfolder ke folder baru.
  
Untuk mempelajari lebih lanjut, baca [menangani folder pesan di Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
