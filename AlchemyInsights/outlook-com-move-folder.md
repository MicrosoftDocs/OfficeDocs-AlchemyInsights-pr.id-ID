# <a name="move-a-folder"></a>Memindahkan folder

Anda dapat menarik folder di bawah **favorit** untuk mengubah urutan mereka.
  
Anda tidak dapat menyusun ulang folder di bawah **folder** atau memindahkan sub-folder untuk menjadi folder.

*Catatan:* Jika Anda memindahkan folder untuk menjadi sub folder, membuat folder baru dan menyalin semua email dari sub-folder ke folder baru.
  
Untuk selengkapnya, lihat [menangani folder pesan di Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
