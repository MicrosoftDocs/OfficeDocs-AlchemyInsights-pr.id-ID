# <a name="cant-delete-files-or-photos-from-files"></a>Tidak dapat menghapus file atau foto dari file

Di Outlook.com, **file** menunjukkan file dan foto yang dilampirkan ke pesan di kotak pesan. Untuk menghapus file dan foto dari file, Anda perlu menghapus pesan email yang dilampirkan. Setelah menghapus pesan, Anda juga harus menghapusnya dari folder Item dihapus.

Untuk mempelajari lebih lanjut, baca [cara menghapus file atau foto di Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d?wt.mc_id=Office_Outlook_com_Alchemy).
