# <a name="cant-delete-files-or-photos-from-files"></a>Tidak dapat menghapus file atau foto dari file

Dalam Outlook.com, **file** menunjukkan file dan foto-foto yang dilampirkan ke pesan di kotak pesan. Untuk menghapus file dan foto dari file, Anda perlu untuk menghapus pesan email mereka melekat. Setelah menghapus pesan, Anda juga perlu untuk menghapusnya dari folder Item dihapus.

Untuk selengkapnya, baca [bagaimana untuk menghapus file atau foto di Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d?wt.mc_id=Office_Outlook_com_Alchemy).
