# <a name="solutions-for-unsupported-operating-system"></a>Solusi untuk "Sistem operasi tidak didukung"



Ketika Anda mencoba untuk menginstal Office 365, Anda mungkin melihat pesan galat **tidak didukung sistem operasi** jika perangkat Anda tidak memenuhi [persyaratan sistem](https://products.office.com/office-system-requirements).
  
Kantor 365 memerlukan:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, atau Windows Server 2008 R2
    
    ATAU
    
- Mac OS X 10.10
    
Jika mendapat pesan ini ketika mencoba untuk menginstal Office pada perangkat mobile atau Chromebook, lihat [cara menginstal dan menjalankan Microsoft Office di Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
