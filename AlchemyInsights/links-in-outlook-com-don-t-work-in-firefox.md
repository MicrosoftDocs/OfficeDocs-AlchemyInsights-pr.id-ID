# <a name="links-in-outlookcom-dont-work-in-firefox"></a>Link di Outlook.com tidak berfungsi di Firefox

Link dalam pesan email Outlook.com tidak berfungsi ketika ditampilkan di browser Firefox. Mozilla sedang [mengupayakan perbaikan](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409) untuk Firefox, namun hingga saat ini:
  
- Gunakan [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) atau browser web lainnya.
- Menampilkan email di panel baca. Untuk informasi selengkapnya, baca cara [mengubah tampilan kotak surat Outlook.com Anda](https://support.office.com/article/b41c2ecb-f23c-42b3-b7f8-659646d5e58c?wt.mc_id=Office_Outlook_com_Alchemy).
