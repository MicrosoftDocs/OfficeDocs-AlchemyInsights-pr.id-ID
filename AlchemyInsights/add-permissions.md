# <a name="add-permissions"></a>Menambahkan izin

1. Gunakan titik akhir izin admin untuk [mengonfigurasi daftar izin yang diminta oleh aplikasi](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application).
1. Ada dua izin yang tersedia untuk memberikan kemampuan untuk membuat pendaftaran aplikasi, masing-masing dengan perilaku berbeda. Untuk informasi selengkapnya, lihat [izin pendaftaran aplikasi untuk peran kustom di Azure Active Directory](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions).
1. [Peran permissions](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions): tabel yang menjelaskan izin tertentu di Azure Active Directory yang diberikan pada setiap peran. Beberapa peran mungkin memiliki izin tambahan di Microsoft Services di luar Azure Active Directory.
