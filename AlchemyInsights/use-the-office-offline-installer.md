# <a name="use-the-office-offline-installer"></a>Gunakan Penginstal offline kantor



Anda harus terhubung ke internet untuk men-download file installer, tetapi setelah selesai, Anda dapat kemudian menginstal Office offline pada PC pada kenyamanan Anda dengan menggunakan alat penyebaran kantor (ODT)
  
ODT dirancang untuk lingkungan perusahaan dan berjalan dari baris perintah, sehingga langkah-langkah lebih rumit--tetapi mereka akan masih bekerja untuk instalasi pada satu perangkat. Lihat [menggunakan kantor offline installer](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA).
