# <a name="use-the-office-offline-installer"></a>Menggunakan Penginstal offline Office

Anda harus tersambung ke internet untuk mengunduh file Penginstal, tapi begitu selesai, Anda bisa menginstal Office offline di PC sesuai keinginan Anda dengan menggunakan Office Deployment Tool (ODT)

ODT didesain untuk lingkungan perusahaan dan berjalan dari baris perintah, jadi langkah-langkahnya lebih rumit--tetapi mereka masih akan berfungsi untuk penginstalan pada satu perangkat. Lihat [menggunakan Penginstal offline Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA).
