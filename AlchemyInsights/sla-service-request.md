# <a name="credit-request-due-to-a-service-outage"></a>Permintaan kredit karena layanan terputus

Untuk mengajukan permohonan kredit karena layanan terputus, silakan buka permintaan layanan.

Pastikan untuk memiliki ID insiden, ditemukan di [halaman Kesehatan Layanan](https://docs.microsoft.com/office365/enterprise/view-service-health) di Microsoft 365 Admin Center.

Harap dicatat bahwa, karena permintaan Anda diproses, itu akan mengalami validasi untuk mengamati dampak dari outage.
