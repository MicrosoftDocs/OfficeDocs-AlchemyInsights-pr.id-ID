# <a name="desktop-app-for-sharepoint"></a>Aplikasi desktop untuk SharePoint

Silakan gunakan Mitra Microsoft Private preview terdaftar Anda untuk mendapatkan bantuan dengan aplikasi desktop untuk SharePoint.
