# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>Kapan perubahan profil saya disinkronkan ke aplikasi profil pengguna SharePoint?

Sinkronisasi profil gambar di SharePoint online memerlukan tindakan pengguna untuk memulai sinkronisasi foto mereka. Tidak ada sinkronisasi otomatis foto ke SharePoint online. SharePoint online hanya menyinkronkan foto dari Exchange Online, tidak Azure Active Directory.

Jika pengguna tidak pernah melakukan tindakan yang akan meminta sinkronisasi atau mereka telah memperbarui gambar profil mereka dan yang baru tidak muncul di SharePoint online, pengguna mungkin perlu meminta untuk disinkronkan. Berikut adalah cara mereka dapat melakukan hal ini:

1. Buka browser web dalam sesi in-Private atau Incognito dan [buka Office.com](http://www.office.com/).

2. Masuk dengan akun pekerjaan atau sekolah mereka, dan klik ubin OneDrive atau SharePoint.

3. [Menambahkan foto profil mereka ke Office 365](https://support.office.com/article/Add-your-profile-photo-to-Office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7).

[Info tentang profil gambar sinkronisasi di Office 365](https://support.office.com/article/Information-about-user-profile-synchronization-in-SharePoint-Online-177eb196-5887-43c9-84c3-b98a43d35129)
