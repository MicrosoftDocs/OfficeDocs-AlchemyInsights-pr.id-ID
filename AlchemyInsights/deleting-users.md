# <a name="deleting-users"></a>Menghapus pengguna

Jika Anda ingin segera memblokir akses pengguna dan perlu untuk menyimpan informasi mereka, silakan lihat [blok mantan karyawan akses ke Office 365 data](https://docs.microsoft.com/office365/admin/add-users/remove-former-employee#block-a-former-employees-access-to-office-365-data).
  
Jika Anda hanya ingin menghapus pengguna dan tidak peduli dengan data mereka, silakan lihat [menghapus pengguna dari organisasi Anda](https://docs.microsoft.com/office365/admin/add-users/delete-a-user). Ini akan segera membebaskan lisensi mereka.
