# <a name="gatewaynotreachable-when-setting-credentials"></a>Gatewaynoterlipat saat mengatur kredensial

Anda mungkin menjumpai Gatewaynoterlipat saat mencoba mengatur kredensial untuk sumber data di dalam Layanan Power BI. Ini bisa berupa hasil Gateway usang. Instal Gateway terbaru dan coba lagi.

Untuk informasi selengkapnya: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
