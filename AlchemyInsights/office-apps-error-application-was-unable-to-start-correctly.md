# <a name="office-apps-error-the-application-was-unable-to-start-correctly"></a>Galat aplikasi Office: aplikasi tidak dapat memulai dengan benar

Untuk memperbaiki galat ini, cobalah opsi berikut ini:

- Verifikasi atau [instal pemutakhiran terbaru](https://support.office.com/article/update-office-and-your-computer-with-microsoft-update-2ab296f3-7f03-43a2-8e50-46de917611c5) untuk Office
- Menyelidiki kemungkinan [masalah dengan Add-in yang](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d?ocmsassetID=HA104114659&CorrelationId=98329f6f-f51f-4f44-a876-4142c3583312#bkmk_addins) diinstal untuk Office
- [Perbaiki Office](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) menggunakan panel kontrol
- [Membongkar](https://support.office.com/article/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8) dan menginstal ulang Office


Untuk informasi selengkapnya, lihat: 

- [Galat saat membuka aplikasi Microsoft Office](https://support.office.com/article/error-when-opening-microsoft-office-apps-b84b6a63-4b8c-46ec-ae9a-ad91d6160d72)
- [Saya mendapatkan kesalahan "berhenti bekerja" ketika saya memulai aplikasi Office di PC saya](https://support.office.com/article/i-get-a-stopped-working-error-when-i-start-office-applications-on-my-pc-52bd7985-4e99-4a35-84c8-2d9b8301a2fa)
- [Memecahkan masalah yang terjadi saat Anda memulai atau menggunakan aplikasi Office](https://docs.microsoft.com/office/troubleshoot/word/issues-when-start-or-use-word)
