# <a name="in-app-support-in-outlook-for-mac"></a>Dukungan dalam aplikasi di Outlook untuk Mac

Untuk menerima dukungan dalam aplikasi untuk masalah Outlook untuk Mac, Buka Outlook dan klik **bantuan** \> **Hubungi dukungan**. Ketikkan pertanyaan Anda, dan anggota tim dukungan Outlook untuk Mac akan menghubungi Anda. 

Pelajari selengkapnya: [Hubungi dukungan di Outlook untuk Mac](https://support.office.com//article/d0410177-8e65-4487-93f7-206a3a3d71a8)
