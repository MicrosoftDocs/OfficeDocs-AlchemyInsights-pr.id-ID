# <a name="copy-and-paste-in-outlookcom"></a>Copy dan paste di Outlook.com

Tidak ada salinan atau sisipkan tombol di Outlook.com, namun Anda dapat menggunakan cara pintas keyboard:

**Pada PC:** CTRL + C untuk menyalin, CTRL + X untuk memotong, dan CTRL + V untuk menyisipkan.

**Pada Mac:** CMD + C untuk menyalin, CMD + X untuk memotong, dan CMD + V untuk menyisipkan.
