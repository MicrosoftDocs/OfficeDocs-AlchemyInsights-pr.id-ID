# <a name="unable-to-open-documents-from-sharepoint-or-onedrive"></a>Tidak dapat membuka dokumen dari SharePoint atau OneDrive

Jika tidak dapat membuka file atau dokumen di situs SharePoint atau OneDrive, mungkin ada masalah layanan sementara ketika pengguna mengalami keterlambatan atau kesalahan navigasi saat mengakses situs SharePoint atau konten OneDrive. Lihat [dasbor Kesehatan Layanan](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) untuk melihat apakah organisasi Anda terpengaruh.
