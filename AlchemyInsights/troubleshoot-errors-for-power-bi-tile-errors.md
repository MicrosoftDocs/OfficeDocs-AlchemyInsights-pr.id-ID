# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Pemecahan masalah skenario refresh - ubin kesalahan

Jika Anda mengalami ubin refresh kesalahan dalam Layanan BI daya, tunggu sekitar 10-15 menit untuk refresh untuk tercermin dalam dashboard ubin. Jika itu masih tidak muncul, kembali pin visualisasi untuk dashboard.

Untuk informasi lebih lanjut:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
