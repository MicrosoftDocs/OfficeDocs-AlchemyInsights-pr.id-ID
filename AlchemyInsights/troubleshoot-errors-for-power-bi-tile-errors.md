# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Pemecahan masalah refresh skenario-ubin kesalahan

Jika Anda mengalami galat refresh ubin dalam Layanan Power BI, Silakan tunggu sekitar 10-15 menit untuk menyegarkan untuk dicerminkan di ubin dasbor. Jika masih belum muncul, Re-pin visualisasi ke dashboard.

Untuk informasi lebih lanjut:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
