# <a name="adblock"></a>AdBlock

Beberapa pengguna saat ini mengalami masalah saat membaca atau menulis email ketika menggunakan Google Chrome dengan ekstensi AdBlock. Untuk mengatasi masalah ini:

1. Menggunakan [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) atau web browser lain.
1. Mengecualikan Outlook.com dari pengaturan AdBlock.Di pojok kanan atas Google Chrome, pilih **AdBlock** dan kemudian pilih **Jangan lari pada halaman pada domain ini**.Refresh browser untuk memuat Outlook.com lagi.
