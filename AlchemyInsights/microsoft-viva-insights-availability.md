# <a name="microsoft-viva-insights-availability"></a>Ketersediaan wawasan Microsoft Viva

Microsoft Viva Insights tersedia sekarang dalam pratinjau.

[Unduh](https://aka.ms/InsightsDocumentation) Viva Insights di Microsoft teams.

[Pelajari selengkapnya](https://aka.ms/VivaInsights) tentang bagaimana Viva Insights membantu menciptakan budaya tempat orang dan bisnis dapat berkembang.

Untuk informasi selengkapnya, lihat bagaimana [Microsoft Viva Insights membantu orang memelihara kesejahteraan dan menjadi yang terbaik](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-insights-helps-people-nurture-wellbeing-and-be/ba-p/2107010).
