# <a name="change-your-profile-name"></a>Mengubah nama profil Anda

1. Buka [Info Akun Microsoft Anda](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Pilih **Edit nama**. 
    
3. Masukkan nama depan dan nama belakang, lalu pilih **Simpan**. 
    
Perbolehkan hingga 24 jam agar perubahan pada nama profil Anda diterapkan.
  
Untuk mempelajari selengkapnya, lihat [mengubah foto profil atau nama Anda di Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
