# <a name="share-a-form"></a>Berbagi bentuk

Untuk berbagi kuis atau bentuk dalam Microsoft Forms, lihat [berbagi bentuk untuk berkolaborasi](https://support.office.com/article/Share-a-form-to-collaborate-d5bb5cf0-8401-4c15-bb8c-8e108cd7e69b).

Untuk mengelola kemampuan untuk berbagi bentuk eksternal, lihat [mengatur bentuk Microsoft](https://support.office.com/article/set-up-microsoft-forms-cc52287a-4550-464d-9a1b-457bf9df2240?ui=en-US&rs=en-US&ad=US#PickTab=Configure). 

Untuk mendapatkan link ringkasan berbagi hasil dengan orang lain:

1. Pada tab **tanggapan** , klik tombol elips (****...).
3. Pilih **Buat link ringkasan**.
4. Klik **Salin**.

Orang dengan siapa Anda berbagi link ini dapat melihat ringkasan dari tanggapan untuk formulir.
