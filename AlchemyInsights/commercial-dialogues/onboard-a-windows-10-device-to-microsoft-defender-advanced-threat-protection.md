# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Onboard perangkat Windows 10 untuk Microsoft Defender Advanced Threat Protection (MDATP)

MDATP mendukung alat dan metode penyebaran berikut ini untuk Windows 10:

- Kebijakan grup
- Microsoft Endpoint Configuration Manager (Manajer Konfigurasi Endpoint Microsoft)
- Manajemen perangkat seluler, termasuk Microsoft Intune
- Skrip lokal

Untuk mempelajari selengkapnya, lihat [alat dan metode Onboarding untuk perangkat Windows 10](https://go.microsoft.com/fwlink/?linkid=2143460).
