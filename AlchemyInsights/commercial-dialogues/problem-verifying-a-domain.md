# <a name="problem-verifying-a-domain"></a>Masalah verifikasi domain

Jika Office 365 tidak memverifikasi domain Anda dalam satu atau dua jam, ada masalah.

Pilih **tidak** pada perintah berikut untuk menunjukkan bahwa **saya tidak menyelesaikan masalah Anda** dan Anda akan melihat opsi untuk berbicara dengan manusia.
