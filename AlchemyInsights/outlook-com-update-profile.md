# <a name="change-my-profile-picture-or-name-in-outlookcom"></a>Mengubah foto profil atau nama saya di Outlook.com

1. Buka [Info Akun Microsoft Anda](https://go.microsoft.com/fwlink/p/?linkid=860841).
1. Pilih **Edit nama** atau **Ubah gambar**.

Perbolehkan hingga 24 jam agar perubahan pada foto profil atau nama Anda diberlakukan.
