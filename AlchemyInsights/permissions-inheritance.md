# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Cara kerja pewarisan izin di SharePoint

Secara default, izin di SharePoint diwarisi lebih tinggi dalam hierarki. Jadi file mewarisi izinnya dari folder, yang mewarisi izinnya dari Pustaka, yang mewarisi izinnya dari situs, yang mewarisi izinnya dari kumpulan situs.
  
Untuk informasi tentang menghapus izin unik dan memulihkan warisan, lihat [mengedit dan mengelola izin untuk daftar atau pustaka](https://go.microsoft.com/fwlink/?linkid=869946).
