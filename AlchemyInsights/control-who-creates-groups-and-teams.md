# <a name="control-who-creates-microsoft-365-groups-and-microsoft-teams"></a>Kontrol yang membuat Microsoft 365 grup dan Microsoft teams

Secara default, semua pengguna dalam organisasi dapat membuat grup Microsoft 365 dan Microsoft teams. Anda dapat mengubah default dan menetapkan hak untuk membuat grup Microsoft 365 atau teams ke subset pengguna dalam organisasi Anda. Perubahan ini juga berdampak membuat grup dari layanan Microsoft 365 seperti Outlook, teams, perencana, dan sebagainya.

Untuk informasi lebih lanjut, lihat [mengelola siapa yang dapat membuat microsoft 365 grup](https://support.office.com/article/Manage-who-can-create-Office-365-Groups-4c46c8cb-17d0-44b5-9776-005fced8e618) dan [Microsoft teams](https://aka.ms/rtsf).
