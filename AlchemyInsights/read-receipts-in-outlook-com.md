# <a name="read-receipts-in-outlookcom"></a>Tanda terima baca di Outlook.com

Anda tidak dapat meminta tanda terima baca di Outlook.com, tetapi Anda dapat meminta tanda terima baca untuk pesan Outlook.com yang dikirim menggunakan Outlook untuk Windows. Untuk mempelajari caranya, lihat [Add dan permintaan tanda terima baca dan pengiriman pemberitahuan](https://go.microsoft.com/fwlink/p/?linkid=874355).
  
Untuk memilih bagaimana Outlook.com menanggapi permintaan tanda terima baca, pergi ke [pesan penanganan setting](https://go.microsoft.com/fwlink/?linkid=2080838) (**pengaturan** > **Lihat semua pengaturan Outlook** > **pesan penanganan**).
