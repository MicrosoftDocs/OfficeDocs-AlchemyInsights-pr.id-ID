# <a name="payment-methods-for-microsoft-for-business"></a>Metode pembayaran untuk Microsoft untuk bisnis

Ketika Anda membeli Microsoft 365 untuk langganan bisnis langsung dari Microsoft, Anda dapat membayar dengan kartu kredit atau debit. Di beberapa negara atau wilayah, Anda juga dapat membayar dengan rekening bank.
  
Jika Anda memilih untuk menggunakan kartu kredit sebagai metode pembayaran, Anda dapat mempelajari cara mengelola dan memecahkan masalah dari [mengelola metode pembayaran](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/manage-payment-methods).
  
Artikel ini juga mencakup saran pemecahan masalah ketika kartu kredit Anda ditolak dan informasi tentang mengubah informasi kartu kredit sehingga kartu diterima.
