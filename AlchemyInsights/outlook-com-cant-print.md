# <a name="unable-to-print"></a>Tidak dapat mencetak

Kami mengetahui masalah yang menyebabkan beberapa pengguna tidak dapat mencetak saat menggunakan browser Safari atau Edge. Saat kami mengerjakan perbaikan untuk ini, silakan gunakan browser lain. Kami Mohon maaf atas ketidaknyamanan ini.
