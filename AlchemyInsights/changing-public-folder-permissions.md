# <a name="changing-public-folder-permissions"></a>Mengubah izin folder publik

Anda dan pengguna Anda dapat mengubah hak akses folder publik di Outlook. Lihat [menetapkan atau mengubah permission untuk folder publik](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005). Admin juga dapat mengubah izin di Exchange admin center:
  
- Di Office 365 admin center, pergi ke **Admin pusat** \> **asing**.
    
- Memilih **folder publik**.
    
Dari sana, Anda dapat mengubah permission untuk folder publik individu dengan menetapkan pengguna grup aman untuk izin. Untuk mengubah permission untuk folder publik, Anda perlu untuk menjadi pemilik itu.
