# <a name="troubleshooting-issues-with-calendar-sharing"></a>Memecahkan masalah dengan berbagi kalender

- Untuk mendiagnosis masalah izin, gunakan [Exchange remote Connectivity Analyzer gratis](https://testconnectivity.microsoft.com/Default.aspx?testId=freeBusy), yang memberi pesan kesalahan yang sangat spesifik.
- Anda juga dapat menggunakan [asisten dukungan dan pemulihan Microsoft untuk microsoft 365](https://diagnostics.office.com/) guna mendiagnosis berbagai masalah kalender lainnya. 
- Jika pengguna memiliki Outlook desktop, [alat pemeriksaan kalender untuk Outlook](https://www.microsoft.com/download/details.aspx?id=28786) dapat mengatasi masalah potensial.

Sumber daya cara-untuk tambahan:

- [Berbagi kalender Outlook dengan orang lain](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)
- [Berbagi kalender Anda di Outlook di web untuk bisnis](https://support.office.com/article/7ecef8ae-139c-40d9-bae2-a23977ee58d5)
