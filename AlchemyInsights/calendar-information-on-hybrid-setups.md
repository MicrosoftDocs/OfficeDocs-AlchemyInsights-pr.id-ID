# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>Kalender bebas/sibuk masalah dalam lingkungan hibrida

Jika pengguna mengalami masalah saat melihat kalender yang bebas/sibuk informasi di lingkungan hibrida, kembali menjalankan wizard konfigurasi hibrida. Untuk informasi lebih lanjut tentang wizard konfigurasi hibrida, lihat [wizard konfigurasi hibrida](https://go.microsoft.com/fwlink/p/?linkid=528149).

Untuk langkah dalam lingkungan hibrida pemecahan masalah, lihat [mengatasi masalah penggunaan hibrid](https://technet.microsoft.com/library/jj659053.aspx).
