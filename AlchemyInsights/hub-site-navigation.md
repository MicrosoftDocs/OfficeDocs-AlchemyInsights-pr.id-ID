# <a name="sharepoint-hub-site-navigation"></a>Navigasi situs hub SharePoint

Di bagian atas situs hub dan situs yang terkait, di atas navigasi situs, adalah bilah navigasi situs hub. Jika Anda adalah pemilik situs hub, Anda dapat menyesuaikan [bar navigasi](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav). 

**Catatan:** Disarankan untuk memilih situs komunikasi, atau situs tim menggunakan template modern sebagai situs hub Anda. Jika Anda menggunakan situs tim klasik, navigasi hub akan muncul hanya pada halaman modern, dan pengaturan situs hub hanya akan muncul di halaman modern.
