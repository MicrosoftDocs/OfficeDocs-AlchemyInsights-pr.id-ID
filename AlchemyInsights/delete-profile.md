# <a name="delete-a-profile"></a>Menghapus profil

Untuk menghapus profil, gunakan langkah berikut.
  
1. Di Outlook, pilih **File** \> **pengaturan** \> akun file **mengelola profil**.

2. Pilih **Tampilkan profil**.

3. Di dialog **mail** , pilih profil yang ingin Anda hapus, lalu klik **Hapus**.

    > [!NOTE]
    > Outlook mungkin memperingatkan Anda bahwa menghapus profil akan menghapus data offline. Anda tidak akan kehilangan informasi yang tersimpan di file data Outlook (. pst) atau informasi apa pun yang tersimpan di akun online, misalnya Outlook.com, Exchange, atau akun Microsoft 365.
  
Untuk **Outlook untuk** manajemen profil Mac, lihat [mengelola profil atau identitas di Outlook untuk Mac](https://support.office.com/article/fed2a955-74df-4a24-bef6-78a426958c4c.aspx) atau menggunakan langkah-langkah berikut ini.
  
1. Dari Finder, buka folder **aplikasi** .

2. Ctrl + klik atau klik kanan **Microsoft Outlook**, dan kemudian pilih **Tampilkan Paket konten**.

3. Pilih **isi** \> **bersama dukungan**, dan kemudian jalankan **manajer profil Outlook**.
