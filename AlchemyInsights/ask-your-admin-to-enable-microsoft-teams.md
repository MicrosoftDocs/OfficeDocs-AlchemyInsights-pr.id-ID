# <a name="youre-missing-out-ask-your-admin-to-enable-microsoft-teams"></a>Anda ketinggalan! Meminta admin Anda untuk mengaktifkan Microsoft teams

Jika memiliki penyewa EDU, Anda harus mengaktifkan Microsoft Teams secara manual. Untuk melakukannya, masuk ke Pusat admin Microsoft 365 > **dan Add-in > Microsoft teams**. Jika Anda menggunakan pusat admin teams baru, Anda harus menonaktifkan **coba Pusat admin baru**   untuk menemukan **layanan dan Add-in**. 

**Penting**: Anda mungkin harus menunggu hingga 24 jam agar perubahan dapat diterapkan.
