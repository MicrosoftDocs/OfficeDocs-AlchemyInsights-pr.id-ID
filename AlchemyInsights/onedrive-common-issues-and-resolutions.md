# <a name="onedrive-common-issues-and-resolutions"></a>Masalah umum dan resolusi OneDrive

Berikut adalah beberapa topik yang sering diminta terkait masalah OneDrive:

- **Siapkan sinkronisasi onedrive**: [sinkronkan file onedrive for Business Anda](https://go.microsoft.com/fwlink/?linkid=533375) atau [sinkronkan file SharePoint](https://go.microsoft.com/fwlink/?linkid=871666) dengan klien sinkronisasi onedrive untuk Windows.  Menggunakan Mac? Anda juga dapat [menyinkronkan file dengan OneDrive](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) .

- **Memecahkan masalah sinkronisasi onedrive**: jika onedrive tidak menyinkronkan file sesuai keinginan, Anda bisa [memecahkan masalah sinkronisasi onedrive](https://go.microsoft.com/fwlink/?linkid=866431) untuk Windows. Menggunakan Mac? Anda juga dapat [memecahkan masalah sinkronisasi](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756) .
- **Batas OneDrive**: Pelajari tentang [larangan dan batasan](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) yang berlaku untuk nama file dan tipe file saat menyinkronkan dengan onedrive.
- **Penyimpanan onedrive**: Lihat cara [mengubah ruang penyimpanan onedrive pengguna tertentu](https://docs.microsoft.com/onedrive/change-user-storage) atau [mengelola penyimpanan onedrive for Business Anda](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Log audit**: Lihat cara [menggunakan pencarian log audit](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) untuk memeriksa aktivitas file, folder, pengguna, atau izin.
