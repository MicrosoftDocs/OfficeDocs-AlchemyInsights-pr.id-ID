# <a name="onedrive-common-issues-and-resolutions"></a>Masalah umum OneDrive dan resolusi

Sepertinya Anda bertanya OneDrive. Kami dapat membantu memberikan jawaban yang lebih baik jika Anda sebentar ulang kata-kata pertanyaan Anda dengan satu atau dua kalimat-kalimat yang menggambarkan masalah Anda - termasuk kesalahan, fitur yang Anda gunakan, dll...  

Sementara itu, berikut adalah beberapa topik yang sering diminta:

- **Setup OneDrive Sync**: [Sync Anda OneDrive untuk bisnis file](https://go.microsoft.com/fwlink/?linkid=533375) atau [sinkronisasi file SharePoint](https://go.microsoft.com/fwlink/?linkid=871666) dengan OneDrive sync client untuk Windows.  Menggunakan Mac? Anda dapat [sinkronisasi file dengan OneDrive](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) dengan itu juga.

- **Mengatasi masalah sinkronisasi OneDrive**: jika OneDrive tidak menyinkronkan file seperti yang Anda harapkan, Anda dapat [memecahkan masalah sinkronisasi OneDrive](https://go.microsoft.com/fwlink/?linkid=866431) untuk Windows. Menggunakan Mac? Anda dapat [memecahkan masalah sinkronisasi](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&rs=en-US&ad=US) untuk itu juga.
- **Batas-batas OneDrive**: belajar tentang [larangan dan pembatasan](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) yang berlaku untuk nama file dan jenis file saat sinkronisasi dengan OneDrive.
- **OneDrive penyimpanan**: Lihat bagaimana untuk [mengubah ruang penyimpanan OneDrive pengguna tertentu](https://docs.microsoft.com/onedrive/change-user-storage) atau [mengelola OneDrive Anda untuk bisnis penyimpanan](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Log audit**: Lihat bagaimana Anda dapat [menggunakan pencarian log Audit](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) untuk memeriksa aktivitas file, folder, pengguna, atau izin.
