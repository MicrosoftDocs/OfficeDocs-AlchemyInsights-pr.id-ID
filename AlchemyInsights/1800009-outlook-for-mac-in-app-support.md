# <a name="in-app-support-in-outlook-for-mac"></a>Di-app dukungan dalam Outlook untuk Mac

Untuk menerima dukungan di-app untuk Outlook untuk Mac masalah, Buka Outlook dan klik bantuan \> Hubungi dukungan. Ketik pertanyaan Anda, dan anggota Outlook untuk Mac tim dukungan akan menghubungi Anda. [Hubungi dukungan Outlook untuk Mac](https://go.microsoft.com/fwlink/?linkid=2002400&amp;clcid=0x409)
