# <a name="install-and-configure-onedrive"></a>Menginstal dan mengkonfigurasi OneDrive

OneDrive sync klien yang terinstal dengan Windows 10. Untuk menginstal OneDrive untuk Windows 7, Mac, atau Linux, atau untuk mendapatkan aplikasi mobile untuk ponsel atau tablet Anda, pergi ke [http://onedrive.com/download](http://onedrive.com/download).
  
Untuk mengatur klien sinkronisasi OneDrive di Windows, lihat:
  
- [Mengatur komputer Anda untuk melakukan sinkronisasi Anda OneDrive untuk bisnis file di Office 365](https://go.microsoft.com/fwlink/?linkid=533375)
    
- [Sinkronisasi SharePoint file dengan OneDrive sync klien baru](https://go.microsoft.com/fwlink/?linkid=871666)
