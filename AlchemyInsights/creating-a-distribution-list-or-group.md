# <a name="create-distribution-groups"></a>Membuat grup distribusi

Anda menggunakan grup distribusi (juga dikenal sebagai daftar distribusi) untuk mengirimkan pesan email ke grup orang di dalam dan di luar organisasi Anda. Membuatnya mudah! Untuk instruksi, lihat [membuat daftar distribusi di pusat admin Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/setup/create-distribution-lists).

**Catatan**: admin harus mengelola keanggotaan grup distribusi. Jika Anda ingin agar pengguna mengelola daftar keanggotaan grup mereka sendiri, Anda mungkin ingin mempertimbangkan [grup Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2) .
