# <a name="getting-an-exchange-hybrid-key"></a>Mendapatkan kunci hibrid Exchange

Untuk menyambungkan organisasi lokal Anda ke Exchange Online di Microsoft 365, Anda harus menyiapkan setidaknya satu server Exchange di organisasi di tempat. Server ini digunakan untuk membuat penggunaan hibrid yang menyambungkan organisasi Exchange lokal ke Exchange Online. Untuk mendapatkan kunci hibrid untuk server ini, ikuti langkah-langkah di [panduan distribusi kunci produk hibrid Exchange](https://aka.ms/hybridkey).
