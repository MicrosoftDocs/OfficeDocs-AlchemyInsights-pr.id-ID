# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Pemecahan masalah penyetelan Outlook Mobile untuk kotak surat bersama

Anda tidak dapat mengakses kotak surat bersama dari aplikasi Outlook yang terinstal di perangkat seluler. Apakah Anda menginginkan fitur ini? Biarkan tim teknik Outlook tahu. [Pilih di sini](https://go.microsoft.com/fwlink/?linked=862116)!
  
Anda hanya bisa menggunakan kotak surat bersama di perangkat seluler dengan membuka browser Anda, masuk, lalu masuk ke Outlook di web. Untuk instruksi, lihat [menambahkan kotak surat bersama di Outlook di web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Dari Outlook di web, Anda dapat mengakses kotak surat bersama.
  
Pelajari selengkapnya: [mendapatkan bantuan dalam aplikasi untuk Outlook untuk IOS dan Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
