# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Pemecahan masalah Outlook Mobile setup untuk kotak surat bersama

Anda tidak dapat mengakses kotak surat bersama dari aplikasi Outlook yang diinstal di perangkat seluler. Apakah Anda ingin fitur ini? Biarkan tim Outlook Engineering tahu. [Suara di sini](https://go.microsoft.com/fwlink/?linked=862116)!
  
Anda hanya dapat menggunakan kotak surat bersama pada perangkat mobile dengan membuka browser Anda, masuk ke Office 365, dan kemudian pergi Outlook di web. Untuk petunjuk, lihat [menambahkan kotak surat bersama di Outlook di web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Dari Outlook di web, Anda dapat mengakses kotak pesan bersama.
  
Pelajari lebih lanjut: [Dapatkan bantuan dalam aplikasi untuk Outlook untuk IOS dan Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
