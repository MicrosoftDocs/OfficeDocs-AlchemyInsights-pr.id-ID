# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Menggunakan SaRA untuk mendiagnosa dan mengatasi masalah otentikasi Outlook

Kami sangat menyarankan Anda menggunakan [Outlook terus meminta password saya](https://aka.ms/SaRA-OutlookPwdPrompt) diagnostik pada mesin terkena untuk memecahkan masalah mana Outlook terus meminta password. Ini [SaRA](https://diagnostics.office.com/#/) diagnostik melakukan pemeriksaan otomatis dan mengembalikan solusi yang mungkin untuk Anda gunakan untuk mengatasi masalah yang terdeteksi.
