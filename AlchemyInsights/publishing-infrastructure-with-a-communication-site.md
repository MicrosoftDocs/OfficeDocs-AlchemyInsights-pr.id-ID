# <a name="publishing-infrastructure-with-a-communication-site"></a>Menerbitkan infrastruktur dengan situs komunikasi


Mengaktifkan fitur infrastruktur penerbitan server SharePoint **tidak didukung** dengan situs komunikasi. 
  
Untuk informasi selengkapnya, lihat [berpindah dari situs penerbitan ke situs komunikasi](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
