# <a name="publishing-infrastructure-with-a-communication-site"></a>Penerbitan infrastruktur dengan sebuah situs komunikasi

Mengaktifkan fitur SharePoint Server Publishing infrastruktur adalah **tidak didukung** dengan situs komunikasi. Harap berikan umpan balik Anda di [UserVoice](https://go.microsoft.com/fwlink/?linkid=2047322&amp;clcid=0x409). 
  
Lihat [mengaktifkan fitur penerbitan](https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518) untuk informasi lebih lanjut.
