# <a name="migrating-to-microsoft-intune"></a>Migrasi ke Microsoft Intune

Jika Anda mencoba untuk bermigrasi ke Intune meninjau [Intune migrasi panduan](https://docs.microsoft.com/intune/fundamentals/migration-guide).

Untuk panduan lebih spesifik tentang migrasi dari System Center Configuration Manager (dan SCCM Hybrid) ke Intune, reivew kami [dokumentasi migrasi hibrid](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa).
