# <a name="error-when-a-microsoft-365-trial-user-sends-email-client-host-blocked-using-spamhaus"></a>Kesalahan ketika pengguna uji coba Microsoft 365 mengirimkan email: host klien diblokir menggunakan SpamHaus

Tampaknya Anda mengirim email dari organisasi percobaan Microsoft 365 baru. Untuk membantu mencegah penyalahgunaan, kami memblokir alamat IP yang sebelumnya telah dikompromikan. Pemeriksaan tertentu ini tidak lagi diberlakukan setelah Anda mulai menggunakan lisensi berbayar. Anda mungkin bisa mendapatkan bantuan sementara dengan meminta alamat IP baru dari penyedia layanan internet (ISP), atau dengan menghapus blok dari situs web [SpamHaus](https://go.microsoft.com/fwlink/p/?linkid=123245) .
