# <a name="discard-a-check-out-from-a-document-library"></a>Buang cek keluar dari pustaka dokumen

1. Klik kanan file yang ingin Anda buang Check Out.
    
2. Klik **lainnya**, dan kemudian klik **buang Check-Out**. 
    
3. Dari kotak dialog, klik **buang Check Out** untuk menghapus Check Out. 
    
Untuk pertanyaan tambahan tentang Check-in dan Check-Out, lihat [pertanyaan teratas tentang pemeriksaan, Check-In, dan versi](https://go.microsoft.com/fwlink/?linkid=2018786).
