# <a name="discard-a-check-out-from-a-document-library"></a>Membuang keluar dari pustaka dokumen

1. Klik kanan file yang Anda ingin membuang memeriksa.
    
2. Klik **lain**, dan kemudian klik **Membuang Check Out**. 
    
3. Dari kotak dialog, klik ** membuang Check Out ** untuk membersihkan cek. 
    
Untuk pertanyaan tentang memeriksa dan memeriksa, lihat [atas pertanyaan tentang memeriksa, check in dan versi](https://go.microsoft.com/fwlink/?linkid=2018786)
