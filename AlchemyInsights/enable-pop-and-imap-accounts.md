# <a name="pop-and-imap4-settings"></a>Pengaturan POP dan IMAP4

Untuk menemukan pengaturan server POP3 dan IMAP4 untuk kotak pesan Anda, lihat [pengaturan email pop dan IMAP untuk Outlook](https://support.office.com/article/8361e398-8af4-4e97-b147-6c6c4ac95353.aspx).

Administrator dapat mengaktifkan dan mengkonfigurasi POP3 dan IMAP4 di organisasi Exchange lokal. Untuk informasi selengkapnya, lihat [mengaktifkan dan MENGKONFIGURASI POP3 di server Exchange](https://technet.microsoft.com/library/bb124934.aspx) dan [mengaktifkan dan mengkonfigurasi IMAP4 di server Exchange](https://technet.microsoft.com/library/bb124489.aspx).
