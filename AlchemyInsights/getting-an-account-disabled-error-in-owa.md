# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Mendapatkan kesalahan akun dinonaktifkan di Outlook di web?

Jika Anda mendapatkan kesalahan **akun Anda dinonaktifkan**  `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` ketika Anda mencoba membuka Outlook di web (sebelumnya dikenal sebagai Outlook Web App), admin organisasi Anda mungkin telah menonaktifkan akses Anda ke Outlook di web. Admin dapat memperbaiki kesalahan ini dengan mengikuti langkah-langkah dalam topik [mengaktifkan atau menonaktifkan Outlook Web App untuk kotak surat](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
