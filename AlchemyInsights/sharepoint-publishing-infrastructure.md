# <a name="sharepoint-publishing-infrastructure"></a>Penerbitan infrastruktur SharePoint


<p>SharePoint mencakup seperangkat klasik fitur yang memungkinkan Anda untuk penulis dan mempublikasikan halaman web yang kaya untuk intranet. Aktifkan SharePoint infrastruktur penerbitan di tingkat situs koleksi untuk mengubahnya.</p>  <p>Untuk informasi lebih lanjut silahkan kunjungi link di bawah ini.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Mengaktifkan fitur penerbitan</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Penerbitan fitur</a></li>  </ul>  <p>Catatan:</p>  <ul>  <li>Template situs penerbitan klasik termasuk fitur penerbitan secara default.</li>  <li>Klasik penerbitan fitur tidak tersedia pada situs modern. Situs modern menyediakan alternatif yang lebih baru secara default.</li>  </ul>
