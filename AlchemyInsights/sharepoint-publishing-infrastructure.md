# <a name="sharepoint-publishing-infrastructure"></a>Infrastruktur penerbitan SharePoint


SharePoint mencakup seperangkat fitur klasik yang memungkinkan Anda untuk menulis dan mempublikasikan halaman web yang kaya untuk Intranet Anda. Mengaktifkan SharePoint penerbitan infrastruktur di tingkat koleksi situs untuk mengaktifkannya.

Untuk informasi lebih lanjut silahkan kunjungi link di bawah ini.

[Mengaktifkan fitur penerbitan](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Fitur publikasi](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Catatan:

- Template situs penerbitan klasik mencakup fitur penerbitan secara default.

- Fitur penerbitan klasik tidak tersedia di situs modern. Situs modern menyediakan alternatif yang lebih baru secara default.
