# <a name="sharepoint-publishing-infrastructure"></a>Penerbitan infrastruktur SharePoint


SharePoint mencakup seperangkat klasik fitur yang memungkinkan Anda untuk penulis dan mempublikasikan halaman web yang kaya untuk intranet. Aktifkan SharePoint infrastruktur penerbitan di tingkat situs koleksi untuk mengubahnya.

Untuk informasi lebih lanjut silahkan kunjungi link di bawah ini.

[Mengaktifkan fitur penerbitan](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Penerbitan fitur](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Catatan:

- Template situs penerbitan klasik termasuk fitur penerbitan secara default.

- Klasik penerbitan fitur tidak tersedia pada situs modern. Situs modern menyediakan alternatif yang lebih baru secara default.
