# <a name="troubleshooting-issues-with-apns-certificate"></a>Pemecahan masalah dengan sertifikat APNs

Jika Anda mengalami masalah dengan sertifikat Layanan pemberitahuan push Apple (APNs), lihat [Intune dan sertifikat APNs: FAQ dan masalah umum](https://techcommunity.microsoft.com/t5/Intune-Customer-Success/Intune-and-the-APNs-certificate-FAQ-and-common-issues/ba-p/280121).

Untuk panduan lebih umum tentang cara memperpanjang sertifikat pemberitahuan push Apple, lihat [mendapatkan sertifikat push Apple Mdm](https://docs.microsoft.com/intune/apple-mdm-push-certificate-get#renew-apple-mdm-push-certificate).
