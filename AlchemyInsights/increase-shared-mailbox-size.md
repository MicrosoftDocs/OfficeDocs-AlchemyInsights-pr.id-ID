# <a name="increase-the-size-of-a-shared-mailbox"></a>Meningkatkan ukuran kotak pesan bersama

Anda dapat meningkatkan ukuran kotak pesan bersama dari ukuran default 50 GB untuk 100 GB dengan menetapkan kotak pesan bersama izin Exchange Online P2 (yang dimasukkan dalam lisensi E3 dan E5). Bila Anda menetapkan lisensi P2 ke kotak pesan bersama di [pengguna aktif](https://portal.office.com/adminportal/home), ukuran akan secara otomatis meningkat dalam waktu 15 menit.
