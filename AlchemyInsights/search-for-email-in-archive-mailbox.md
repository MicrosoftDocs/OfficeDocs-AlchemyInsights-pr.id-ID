# <a name="search-for-email-in-the-archive-mailbox"></a>Cari email di kotak pesan arsip

Bila kotak pesan arsip diaktifkan untuk Office 365, Anda dapat menggunakan **eDiscovery Cari dan ekspor** untuk menemukan konten di kotak pesan utama dan arsip. Untuk informasi tentang menggunakan **eDiscovery Cari ekspor** untuk menemukan konten tersebut, ikuti langkah-langkah diuraikan [di sini](https://docs.microsoft.com/office365/securitycompliance/export-search-results).
  
Anda juga dapat menyimpan konten dengan menambahkan kotak surat ke kebijakan penyimpanan. Ini akan menempatkan konten di kotak pesan utama dan arsip ditahan. Untuk informasi lebih lanjut tentang membuat kebijakan penyimpanan untuk menyimpan email Lihat langkah diuraikan [di sini](https://docs.microsoft.com/Office365/securitycompliance/retention-policies).
