# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Izin delegasi kotak surat dalam lingkungan hibrid

Bergantung pada versi Exchange yang telah Anda instal di organisasi lokal, Anda mungkin perlu melakukan beberapa langkah tambahan untuk menyetel izin delegasi kotak surat antara organisasi lokal dan Exchange Online di Microsoft 365. Untuk informasi selengkapnya tentang cara menyetel izin yang didelegasikan, lihat [mengonfigurasi Exchange untuk mendukung izin kotak surat delegasi dalam penggunaan hibrid](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
