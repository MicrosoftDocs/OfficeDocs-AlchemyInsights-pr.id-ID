# <a name="internal-server-errors-in-outlookcom"></a>Kesalahan server internal di Outlook.com

Oh tidak! Jika Anda melihat "kesalahan 500" atau "kesalahan server internal" pesan, itu adalah kami dan tidak Anda-menyesal tentang hal itu.

Periksa [halaman status layanan Office](https://portal.office.com/servicestatus) untuk memastikan tidak ada yang lain yang terjadi, namun solusi terbaiknya adalah untuk beristirahat selama beberapa menit, lalu refresh browser Anda dengan menekan **Ctrl** dan **F5**.
