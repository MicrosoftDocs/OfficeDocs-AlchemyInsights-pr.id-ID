# <a name="powerpoint-common-issues-and-resolutions"></a>Masalah umum dan resolusi PowerPoint

Berikut adalah beberapa topik yang sering diminta dengan PowerPoint:

1. [PowerPoint tidak merespons, mengalami hang, atau berhenti berfungsi](https://support.office.com/article/PowerPoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d).

2. [PowerPoint tidak dapat menyisipkan video dari file yang dipilih](https://support.office.com/article/PowerPoint-cannot-insert-a-video-from-the-selected-file-acd46430-9e0c-4dca-9484-19cf0afdde7c).

3. **Video**: [menyisipkan file video dari komputer Anda](https://support.office.com/article/insert-and-play-a-video-file-from-your-computer-f3fcbd3e-5f86-4320-8aea-31bff480ed02), [menyisipkan video dari YouTube](https://support.office.com/article/Insert-a-video-from-YouTube-or-another-site-8340ec69-4cee-4fe1-ab96-4849154bc6db), dan [format file video dan audio yang didukung di PowerPoint](https://support.office.com/article/video-and-audio-file-formats-supported-in-powerpoint-d8b12450-26db-4c7b-a5c1-593d3418fb59).

4. **Pencetakan**: untuk praktik terbaik dan masalah terkait pencetakan, Tinjau informasi dalam [mencetak slide dan catatan PowerPoint Anda](https://support.office.com/article/Print-your-PowerPoint-slides-handouts-or-notes-194d4320-aa03-478b-9300-df25f0d15dc4). 

5. **Font**: untuk berbagi font dengan presentasi Anda, baca [menyematkan font di Word atau PowerPoint](https://support.office.com/article/Embed-fonts-in-Word-or-PowerPoint-cb3982aa-ea76-4323-b008-86670f222dbc).

6. **Fitur**: [membandingkan fitur PowerPoint](https://support.office.com/article/Compare-PowerPoint-features-on-different-platforms-90986850-227c-4b25-938e-1c5838166b8b#bm11) di platform Windows, Mac, web, dan seluler.
