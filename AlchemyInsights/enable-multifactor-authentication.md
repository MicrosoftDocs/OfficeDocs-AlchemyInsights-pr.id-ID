# <a name="enable-multi-factor-authentication"></a>Mengaktifkan otentikasi faktor multi

1. Menavigasi ke **pengguna** \> **pengguna aktif**
    
2. Klik **lebih** \> **Setup Azure faktor multi auth**. 
    
3. Cari pengguna atau pengguna yang ingin Anda Aktifkan untuk MFA. Untuk melihat semua pengguna, Anda mungkin perlu mengubah tampilan status Auth multi faktor di atas.
    
4. Periksa kotak centang di sebelah pengguna ingin mengaktifkan.
    
5.  Pada panel info pengguna yang tepat, di bawah **langkah cepat** Anda akan melihat mengaktifkan dan mengelola pengaturan pengguna. Memilih **mengaktifkan**. 
    
6. Di kotak dialog yang terbuka, **klik Aktifkan auth faktor multi**.
