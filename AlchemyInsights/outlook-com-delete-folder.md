# <a name="delete-a-folder-in-outlookcom"></a>Menghapus folder di Outlook.com

1. Di panel kiri, klik kanan folder atau subfolder yang ingin Anda hapus dan pilih **Hapus folder**.
2. Pilih **OK** untuk mengonfirmasi.

Untuk mempelajari selengkapnya, lihat [bekerja dengan folder pesan di Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
