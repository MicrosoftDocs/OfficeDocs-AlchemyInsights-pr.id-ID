# <a name="unable-to-delete-photos-from-files"></a>Tidak dapat menghapus foto dari file

Untuk menghapus file atau foto:
  
1. Pergi ke folder **kotak masuk** atau **Item terkirim** dan menghapus pesan email mereka melekat. 
    
2. Menghapus item dari Anda **Deleted items**. 
    
Untuk informasi selengkapnya, baca [bagaimana untuk menghapus file atau foto di Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx).
