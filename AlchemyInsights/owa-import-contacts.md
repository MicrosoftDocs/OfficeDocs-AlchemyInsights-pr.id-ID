# <a name="import-contacts"></a>Mengimpor kontak

Sebelum mengimpor kontak ke Outlook di web, Anda akan perlu untuk ekspor mereka dari penyedia email lain sebagai .csv file. Kemudian, ikuti petunjuk di bawah.

1. Di bagian bawah halaman, pilih ikon orang untuk pergi ke halaman orang.
2. Pada toolbar, pilih **Kelola** > **Impor kontak**.
3. Pilih **Browse**, pilih CSV file, dan kemudian pilih **buka**.
4. Pilih **impor**.

Untuk selengkapnya, lihat [kontak impor ke Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
