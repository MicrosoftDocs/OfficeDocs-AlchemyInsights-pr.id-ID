# <a name="set-up-microsoft-information-protection"></a>Menyiapkan perlindungan informasi Microsoft

Dapatkan gambaran umum kapabilitas yang dapat diterapkan pada strategi perlindungan informasi sehingga Anda dapat yakin bahwa informasi sensitif Anda telah diproteksi. Gunakan pendekatan siklus hidup empat-tahap yang Anda temukan, klasifikasikan, Lindungi, dan Pantau informasi sensitif. Panduan "penyetelan proteksi informasi Microsoft" menyediakan panduan untuk menyelesaikan setiap tahapan ini.
