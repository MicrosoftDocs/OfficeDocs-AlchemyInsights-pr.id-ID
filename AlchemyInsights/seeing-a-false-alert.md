# <a name="seeing-a-false-alert"></a>Melihat peringatan palsu

- Kirim file ke Microsoft untuk dianalisis. Untuk info lebih lanjut, lihat [apa yang harus dilakukan dengan positif palsu/negatif di Microsoft Defender antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Membuat indikator "Izinkan" untuk mencegah peringatan positif palsu. Untuk informasi lebih lanjut, lihat [membuat indikator untuk file](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Konfigurasikan pengecualian pada perangkat untuk mencegah item dipindai. Untuk informasi lebih lanjut, lihat [menentukan pengecualian pada perangkat Windows individual untuk mencegah item dipindai](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
