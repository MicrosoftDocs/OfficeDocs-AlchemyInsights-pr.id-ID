# <a name="learn-how-to-configure-agenda-settings"></a>Pelajari cara mengonfigurasi setelan agenda

Pengaturan agenda terletak di pengaturan **kalender** di Outlook di web. Untuk informasi selengkapnya, lihat [pengaturan kalender di Outlook di web](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).

Admin dapat menggunakan cmdlet **set-CalendarNotification** di Exchange Online PowerShell untuk mengkonfigurasi pengaturan agenda. Untuk informasi selengkapnya, lihat [set-CalendarNotification](https://technet.microsoft.com/library/dd351284).
