# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Memetakan pustaka SharePoint ke drive jaringan

Memetakan pustaka sebagai drive jaringan bersifat sementara dan hanya didukung melalui Internet Explorer. Anda sesekali harus membuka situs SharePoint di Internet Explorer dan pilih **tetap masuk** untuk mencegah sesi kedaluwarsa. Sebagai gantinya, [sinkronkan file SharePoint dengan klien sinkronisasi OneDrive baru](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> yang menyediakan [file sesuai permintaan](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Mengakses semua file Anda di OneDrive tanpa menggunakan ruang penyimpanan lokal.

Jika Anda memilih untuk memetakan drive daripada [menggunakan klien sinkronisasi OneDrive baru](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), pastikan Anda mengikuti langkah-langkah dalam artikel di bawah ini. 


**Cara mengonfigurasi dan memecahkan masalah drive jaringan yang dipetakan**


Lihat [memecahkan masalah drive jaringan yang dipetakan yang tersambung ke SharePoint Onlines](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives).

Catatan: jika Anda menggunakan Internet Explorer 10 dengan Windows 8 atau Windows 7, dan menerima **Akses ditolak** atau **jalur tidak bisa diakses** saat memetakan drive, instal [perbaikan terbaru ini](https://support.microsoft.com/help/2846960) untuk mengatasi masalah ini.
