# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Peta SharePoint perpustakaan untuk drive jaringan

Perpustakaan sebagai drive jaringan pemetaan sementara dan didukung hanya melalui Internet Explorer. Kadang-kadang Anda harus membuka situs SharePoint di Internet Explorer dan pilih "Tetap masuk" untuk mencegah sesi kedaluwarsa. Sebaliknya, [sinkronisasi file SharePoint dengan klien sinkronisasi OneDrive baru](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> yang menyediakan [File On-Demand](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Mengakses semua file Anda di OneDrive tanpa menggunakan ruang penyimpanan lokal.

Jika Anda memilih untuk memetakan kandar daripada[menggunakan OneDrive sync klien baru](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), pastikan Anda mengikuti langkah-langkah dalam artikel di bawah ini. 


**Cara mengkonfigurasi dan memecahkan masalah dipetakan drive jaringan**


- [Mengkonfigurasi dan memecahkan masalah drive jaringan yang dipetakan](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

Catatan: Untuk Internet Explorer 10 dengan Windows 8 atau Windows 7 yang menerima "Akses ditolak" atau "Jalan ini tidak dapat diakses" ketika pemetaan drive, menginstal perbaikan terbaru ini untuk mengatasi masalah ini. Pergi ke [kesalahan ketika Anda membuka SharePoint pustaka dokumen pada Windows Explorer atau peta drive jaringan ke Perpustakaan setelah Anda menginstal Internet Explorer 10](https://support.microsoft.com/help/2846960).
