# <a name="activating-windows-10-subscriptions"></a>Mengaktifkan langganan Windows 10

Microsoft 365 bisnis Premium, Microsoft 365 pendidikan, Microsoft 365 dan Enterprise memberikan lisensi digital untuk Windows 10 sebagai bagian dari langganan. Untuk mengaktifkan langganan, perangkat Anda harus sudah mengaktifkan Windows 10 Pro (1703 atau yang lebih baru). Untuk melihat daftar lengkap persyaratan untuk mengaktifkan Windows 10 berlangganan Lihat [Windows 10 berlangganan aktivasi persyaratan](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
