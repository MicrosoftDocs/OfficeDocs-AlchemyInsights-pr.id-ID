# <a name="move-files-from-onedrivecom-or-another-provider-into-onedrive-for-business"></a>Memindahkan file dari OneDrive.com atau penyedia lain ke OneDrive untuk bisnis

Jika Anda memiliki akses lokal ke berkas, Anda dapat memindahkan berkas ke OneDrive untuk bisnis dengan meng-upload mereka di web browser atau dengan menggunakan OneDrive sinkronisasi klien. Kami memiliki alat tambahan untuk membantu Anda bergerak:

- Dapatkan ringkasan yang komprehensif tentang cara memindahkan konten ke OneDrive for Business. Untuk informasi lebih lanjut, lihat [memigrasi konten Anda ke Microsoft 365](https://docs.microsoft.com/sharepointmigration/migrate-to-sharepoint-online).
- Anda harus menggunakan langkah yang berbeda untuk memindahkan buku catatan OneNote Anda. Untuk informasi lebih lanjut, lihat [memindahkan buku catatan OneNote ke OneDrive](https://support.office.com/article/move-a-onenote-notebook-to-onedrive-0af0a141-0bdf-49ab-9e50-45dbcca44082).
    
- Mengatur file on demand. Anda dapat menggunakan file on demand untuk Windows dan Mac. Untuk informasi lebih lanjut, lihat [menghemat ruang disk dengan file Onedrive on-demand untuk Windows 10](https://support.office.com/article/Save-disk-space-with-OneDrive-Files-On-Demand-for-Windows-10-0e6860d3-d9f3-4971-b321-7092438fb38e) dan [menghemat ruang disk dengan file Onedrive on-demand untuk Mac](https://support.office.com/article/Save-disk-space-with-OneDrive-Files-On-Demand-for-Mac-529f6d53-e572-4922-a585-e7a318c135f0).
- Cadangkan folder penting (folder desktop, dokumen, dan gambar) di PC Windows Anda dengan pencadangan folder OneDrive PC, sehingga mereka terlindungi dan tersedia di perangkat lain. Untuk info selengkapnya, lihat [mencadangkan folder dokumen, Gambar, dan desktop Anda dengan onedrive](https://support.office.com/article/back-up-your-documents-pictures-and-desktop-folders-with-onedrive-d61a7930-a6fb-4b95-b28a-6552e77c3057) dan [cara onedrive melindungi data Anda di Cloud](https://support.office.com/article/how-onedrive-safeguards-your-data-in-the-cloud-23c6ea94-3608-48d7-8bf0-80e142edd1e1). 
- Saat Anda memindahkan file bersama ke OneDrive for Business, mereka akan dikenali sebagai file baru, jadi Anda mungkin ingin membagikan file lagi. Untuk informasi lebih lanjut, lihat [berbagi berkas di OneDrive untuk bisnis](https://support.office.com/article/Move-files-from-OneDrive-to-OneDrive-for-Business-7fb28cad-7e25-451f-8b4b-2d1a71e5c0e9#sharefiles).
