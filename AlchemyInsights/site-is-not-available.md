# <a name="site-is-not-available"></a>Situs ini tidak tersedia

Jika situs Anda SharePoint atau OneDrive tidak tersedia untuk beberapa pengguna, mungkin ada masalah sementara layanan. Periksa[Layanan kesehatan Dashboard](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) untuk melihat jika penyewa Anda dipengaruhi.
