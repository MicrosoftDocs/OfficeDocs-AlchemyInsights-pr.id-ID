# <a name="create-a-folder"></a>Membuat folder

1. Pada panel kiri, pilih **Buat Folder Baru**![buat tombol folder baru](media/d8e28612-fbdb-4d28-a4d0-14f7834cfd97.png) di sebelah **folder**. 
    
2. Dalam kotak folder baru, masukkan nama folder dan tekan Enter.
    
Untuk selengkapnya, lihat [membuat folder di Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
