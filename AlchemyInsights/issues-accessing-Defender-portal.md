# <a name="issues-accessing-the-microsoft-defender-atp-portal"></a>Masalah mengakses portal Microsoft Defender ATP

Jika Anda menerima pesan "langganan Anda telah kedaluwarsa", langganan layanan online Anda telah kedaluwarsa. Langganan Microsoft Defender ATP, seperti langganan layanan online lainnya, memiliki tanggal kedaluwarsa.

Pesan "langganan Anda telah kedaluwarsa" menyajikan dengan opsi untuk mengunduh paket perangkat off-boarding jika Anda memilih untuk tidak memperpanjang lisensi Anda.
