# <a name="setting-up-a-hybrid-environment"></a>Mendirikan lingkungan hibrida

Untuk mengijinkan user Anda bekerja secara lancar sama tidak peduli jika mereka disimpan di server lokal atau di Office 365, Anda mungkin perlu mengatur lingkungan hibrida. Untuk mendapatkan gambaran tentang pilihan Anda, periksa [Office 365 integrasi dengan lingkungan lokal](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
