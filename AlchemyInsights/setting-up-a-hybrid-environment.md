# <a name="setting-up-a-hybrid-environment"></a>Menyiapkan lingkungan hibrida

Untuk membiarkan pengguna Anda bekerja dengan mulus bersama-sama tidak peduli jika data mereka disimpan di server lokal atau di Microsoft 365, Anda mungkin perlu menyiapkan lingkungan hibrid. Untuk mendapatkan Ikhtisar pilihan Anda, periksa [integrasi 365 Microsoft dengan lingkungan lokal](https://docs.microsoft.com/office365/enterprise/office-365-integration).
