# <a name="setting-up-a-hybrid-environment"></a>Menyiapkan lingkungan hibrida

Untuk mengizinkan pengguna Anda bekerja dengan mulus bersama-sama tidak masalah jika data mereka disimpan di server lokal atau di Office 365, Anda mungkin perlu menyiapkan lingkungan hibrid. Untuk mendapatkan Ikhtisar pilihan Anda, periksa [integrasi 365 Office dengan lingkungan lokal](https://docs.microsoft.com/office365/enterprise/office-365-integration).
