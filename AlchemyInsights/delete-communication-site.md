# <a name="delete-a-communication-site"></a>Menghapus sebuah situs komunikasi

1. Pergi ke situs.
    
2. Klik ikon setelan di bagian kanan atas.
    
3. Klik **pengaturan situs**. Dalam **Situs informasi**, klik **Hapus situs ini**, dan kemudian klik **Hapus**.
