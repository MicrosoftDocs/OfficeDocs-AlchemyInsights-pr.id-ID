# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>Mengkonversi grup distribusi ke Microsoft 365 grup

Anda dapat mengkonversi grup distribusi ke Microsoft 365 grup jika mereka memenuhi kriteria tertentu. Untuk mengonversi grup atau memecahkan masalah konversi, lihat [meningkatkan daftar distribusi ke Microsoft 365 grup di Outlook](https://docs.microsoft.com/office365/admin/manage/upgrade-distribution-lists).
