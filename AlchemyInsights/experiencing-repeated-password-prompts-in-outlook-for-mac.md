# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Mengalami berulang sandi wantian di Outlook untuk Mac?

Kami dapat membantu menyediakan lebih khusus Outlook untuk Mac bantuan langsung dari aplikasi.
  
Silakan pergi ke pengguna yang terkena Mac, Buka Outlook, dan klik pada **bantuan** \> **Hubungi dukungan**.
  
Untuk mempelajari selengkapnya tentang menggunakan Outlook untuk Mac dalam aplikasi dukungan pelanggan, lihat artikel ini: [fitur dukungan kontak baru di Outlook 2016 untuk Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
