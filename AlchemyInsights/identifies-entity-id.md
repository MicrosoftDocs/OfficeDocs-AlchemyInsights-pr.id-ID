# <a name="identifiers-entity-id"></a>Pengidentifikasi (ID entitas)

Untuk menampilkan atau mengedit klaim yang diterbitkan dalam token SAML ke aplikasi, buka aplikasi di Azure portal. Lalu buka bagian **atribut pengguna & klaim** . Untuk informasi selengkapnya, lihat [mengkustomisasi klaim yang dikeluarkan dalam tanda SAML untuk aplikasi perusahaan](https://docs.microsoft.com/azure/active-directory/develop/active-directory-saml-claims-customization#editing-nameid).
