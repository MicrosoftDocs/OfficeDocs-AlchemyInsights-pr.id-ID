# <a name="inbox-not-loading-with-firefox"></a>Inbox yang tidak memuat Firefox

Pengaya tertentu pada Firefox browser mungkin mencegah Outlook.com inbox loading.
  
Kunjungi Dukungan Firefox untuk mempelajari cara [menonaktifkan atau menghapus Pengaya](https://support.mozilla.org/kb/disable-or-remove-add-ons).
