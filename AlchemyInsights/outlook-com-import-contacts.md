# <a name="import-contacts"></a>Mengimpor kontak

Sebelum mengimpor kontak Outlook.com, Anda akan perlu untuk ekspor mereka dari penyedia email lain sebagai .csv file. Kemudian, ikuti petunjuk di bawah.
  
1. Pergi ke [orang-orang](https://outlook.live.com/people/).
2. Di bagian atas halaman, pilih **Kelola** \> **Impor kontak**.
3. Pilih **People**, navigasikan ke folder tempat Anda menyimpan Anda ekspor kontak, lalu pilih **buka**.

Untuk selengkapnya, lihat [Impor kontak Outlook.com](https://support.office.com/article/285a3b55-8d93-4ac8-93df-43fffd13b2f1?wt.mc_id=Office_Outlook_com_Alchemy).
