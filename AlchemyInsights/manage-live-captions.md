# <a name="manage-live-captions"></a>Mengelola subtitel Live

Untuk informasi selengkapnya tentang cara menggunakan fitur keterangan langsung, lihat [menggunakan subtitel langsung di Rapat tim](https://support.microsoft.com/office/use-live-captions-in-a-teams-meeting-4be2d304-f675-4b57-8347-cbd000a21260) atau [menggunakan subtitel Live di acara siaran langsung](https://support.microsoft.com/office/use-live-captions-in-a-live-event-1d6778d4-6c65-4189-ab13-e2d77beb9e2a).  

Secara default, subtitel Live tidak diaktifkan secara otomatis untuk pengguna selama Rapat. Pengguna melihat opsi **Aktifkan subtitel langsung** di menu overflow **(...)** untuk mengaktifkannya. Untuk mengelola atau menonaktifkan subtitel langsung, lihat [mengaktifkan subtitel langsung](https://docs.microsoft.com/microsoftteams/meeting-policies-in-teams#enable-live-captions).

Untuk selengkapnya tentang penerapan kebijakan Rapat, lihat [mengelola kebijakan Rapat di teams](https://docs.microsoft.com/microsoftteams/meeting-policies-in-teams).
