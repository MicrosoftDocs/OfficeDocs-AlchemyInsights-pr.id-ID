# <a name="error-code-30174-4-when-installing-office"></a>Kode kesalahan 30174-4 ketika menginstal Office

Anda mungkin melihat kesalahan ini jika Anda software anti-virus, firewall, pengaturan proxy atau koneksi mencegah Anda dari menginstal Office. Berikut adalah beberapa hal yang Anda dapat mencoba untuk memperbaiki masalah.
  
- Menggunakan sambungan kabel

- Untuk sementara menonaktifkan pengaturan proxy

- Sementara Nonaktifkan perangkat lunak antivirus
