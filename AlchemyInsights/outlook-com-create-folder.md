# <a name="create-a-folder-in-outlookcom"></a>Buat folder di Outlook.com

1. Di bagian bawah panel folder, pilih **folder baru**. Anda mungkin perlu menggulir ke bawah di panel folder untuk melihat tombol **folder baru** .
2. Di kotak teks, masukkan nama untuk folder dan kemudian tekan Enter.

Untuk mempelajari lebih lanjut, lihat [membuat folder di Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
