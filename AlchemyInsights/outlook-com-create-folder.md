# <a name="create-a-folder-in-outlookcom"></a>Membuat folder di Outlook.com

1. Di bagian bawah panel folder, pilih **folder baru**. Anda mungkin perlu gulir ke bawah di panel folder untuk melihat tombol **folder baru** .
2. Dalam kotak teks, masukkan nama folder dan kemudian tekan Enter.

Untuk selengkapnya, lihat [membuat folder di Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
