# <a name="working-with-sharepoint-permission-levels"></a>Bekerja dengan tingkat izin SharePoint

SharePoint dilengkapi dengan tingkat izin default yang dapat Anda gunakan, seperti **kontribusi** dan **hanya melihat**. Tapi jika salah satu dari ini tidak memenuhi kebutuhan Anda, Anda dapat membuat tingkat izin baru. Perlu diingat bahwa lebih mudah untuk melacak tingkat izin jika Anda tidak mengubah default.

[Cara membuat dan mengedit tingkat izin](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
