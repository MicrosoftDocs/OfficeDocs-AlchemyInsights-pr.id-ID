# <a name="working-with-sharepoint-permission-levels"></a>Bekerja dengan tingkat izin SharePoint

SharePoint dilengkapi dengan tingkat izin default yang bisa Anda gunakan, seperti **kontribusi** dan **tampilan saja**. Tapi jika salah satu dari ini tidak memenuhi kebutuhan Anda, Anda bisa membuat tingkat izin baru. Ingatlah bahwa lebih mudah melacak tingkat izin jika Anda tidak mengubah default.

[Cara membuat dan mengedit tingkat izin](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
