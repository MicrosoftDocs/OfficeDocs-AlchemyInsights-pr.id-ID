# <a name="change-the-domain-for-microsoft-365-group"></a>Mengubah domain untuk Microsoft 365 grup

Jika Anda ingin mengubah domain untuk Microsoft 365 grup selama pembuatan, ikuti langkah di [Pilih domain yang digunakan saat membuat Microsoft 365 grup](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups).
  
Setelah Microsoft 365 grup telah dibuat, saat ini tidak ada cara untuk mengubah domain. Satu-satunya cara adalah untuk membuat baru Microsoft 365 grup dan menghapus yang lama jika diperlukan (alamat email tambahan dapat ditambahkan melalui [Pusat admin Exchange](https://outlook.office365.com/ecp.aspx)).
