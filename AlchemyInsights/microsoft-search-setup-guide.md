# <a name="microsoft-search-setup-guide"></a>Panduan penyetelan Microsoft Search

Gunakan [panduan penyetelan Microsoft Search](https://go.microsoft.com/fwlink/?linkid=2153798) untuk menyetel dan mengonfigurasi pencarian Microsoft. Apakah Anda ingin mengemudikan ke grup pengguna atau memindahkannya ke semua orang di organisasi Anda. Anda akan menetapkan admin pencarian dan editor pencarian lalu mengkustomisasi pengalaman pencarian untuk pengguna Anda dengan jawaban dan opsi tambahan, seperti menambahkan ekstensi Bing ke Chrome atau mengatur Bing sebagai mesin pencarian default Anda.
