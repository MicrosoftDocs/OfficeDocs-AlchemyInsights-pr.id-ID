# <a name="help-understanding-value-added-tax-vat"></a>Membantu memahami pajak pertambahan nilai (PPN)

Jika Anda berada di negara yang menggunakan pajak pertambahan (PPN), seperti Inggris, faktur Microsoft menghitung PPN untuk Anda, dan berlaku jumlah faktur Anda.
  
Untuk alasan pajak, Anda juga harus menentukan nomor pendaftaran PPN dalam pengaturan untuk faktur.
  
Anda tidak dapat menambahkan atau menghapus TARIF PPN, namun Anda dapat mengubah TARIF PPN default di setelan. Secara default, faktur menggunakan layanan untuk memvalidasi nomor pendaftaran PPN, tetapi kami sarankan Anda membaca [sanggahan Vies](https://go.microsoft.com/fwlink/?LinkID=841741) diterbitkan oleh Komisi Eropa. Jika Anda memilih untuk menonaktifkan layanan, Anda harus memverifikasi semua nomor pendaftaran PPN.
